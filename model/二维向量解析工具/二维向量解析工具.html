二维向量解析工具

功能

二维向量解析工具用来将一个二维向量数据（scVector）解析成两个double类型数据。其作用与二维向量生成工具相反。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 二维向量 即将要解析的二维向量数据。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 X分量 二维向量解析出的X分量。
Y分量 二维向量解析出的Y分量。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 X分量 二维向量解析出的X分量，供后序工具使用，同监视窗口的X分量参数。
Y分量 二维向量解析出的Y分量，供后序工具使用，同监视窗口的Y分量参数。
