视图清空工具

功能

用来清空所选择的视图，当工程中使用视图清空工具时，工具执行前视图上显示的内容将被清空，工具执行后要显示的内容正常显示。

参数

分类 参数名称 参数描述
属性窗口 视图名称 选择需要清空的视图窗口，比如，View-1等。注意，如果没有View视图，视图名称中没有选项。
延时时长 设定延时时长以避免清空不及时，参考值0-1000，单位：ms。
是否清空图像 控制是否清空图像。
图像窗口 无 无
数据链 无 无
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 无 无
