详细说明

scAnnulusSection()

  • 功能：构造scAnnulus对象。

  • 参数：无。

  • 返回值：无。

scAnnulusSection( center,radius,startPhi,spanPhi,dRadialScale )

功能：用圆心center，基准半径radius，起始角度startPhi，张角spanPhi和径向缩放比例dRadialScale来构造圆环段。

  • 参数：

    • center：圆心位置，sc2Vector类型。

    • radius：基准半径，浮点数类型。

    • startPhi：起始角度，浮点数类型。

    • spanPhi：张角，浮点数类型。

    • dRadialScale：径向缩放比例，浮点数类型。

  • 返回值：无。

scAnnulusSection( annulus, startPhi, spanPhi )

  • 功能：用圆环annulus，起始角度startPhi和张角spanPhi来构造圆环段。

  • 参数：

    • annulus：圆环，scAnnulus类型。

    • startPhi：起始角度，浮点数类型。

    • spanPhi：张角，浮点数类型。

  • 返回值：无。

SetAnnulus( annulus )

  • 功能：设置圆环段所在的圆环。

  • 参数：

    • annulus：圆环，scAnnulus类型。

  • 返回值：bool类型，表示是否设置成功。

GetAnnulus()

  • 功能：获取圆环段所在的圆环。

  • 参数：无

  • 返回值：圆环段所在的圆环，scAnnulus类型。

SetCenter( center )

  • 功能：设置圆环中心。

  • 参数：

    • center：圆环中心，sc2Vector类型。

  • 返回值：bool类型，表示是否设置成功。

GetCenter()

  • 功能：获取圆环段所在的圆环的中心。

  • 参数：无

  • 返回值：圆环段所在的圆环的中心，sc2Vector类型。

SetRadius( radius )

  • 功能：设置圆环的基准半径。

  • 参数：

    • radius：基准半径，浮点数类型。

  • 返回值：bool类型，表示是否设置成功。

GetRadius()

  • 功能：获取圆环的基准半径。

  • 参数：无。

  • 返回值：圆环的基准半径，浮点数类型。

SetRadialScale( dRadialScale )

  • 功能：设置径向缩放比例。

  • 参数：

    -dRadialScale：径向缩放比例，浮点数类型。

  • 返回值：bool类型，表示是否设置成功。

GetRadialScale()

  • 功能：获取径向缩放比例。

  • 参数：无。

  • 返回值：径向缩放比例，浮点数类型。

  • 功能：设置起始角度。
