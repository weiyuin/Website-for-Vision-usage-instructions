scLineSeg

功能

表示线段。

接口

分类 接口名称 接口描述
构造 scLineSeg scLineSeg()默认构造函数。
scLineSeg scLineSeg(sc2Vector, sc2Vector)构造函数。
函数 Set 设置线段起始点和终止点。
SetP1 设置线段起始点。
SetP2 设置线段终止点。
GetP1 获取线段起始点。
GetP2 获取线段终止点。
GetLine 获取当前线段所在的直线。
GetRotation 获取线段的旋转角度。

详细说明

scLineSeg()

  • 功能：构造scLineSeg对象，默认构造函。

  • 参数：无。

  • 返回值：无。

scLine(p1, p2)

  • 功能：用起始点p1和终止点p2构造线段对象。

  • 参数：

    • p1：线段的起始点，sc2Vector类型。

    • p2：线段的终止点，sc2Vector类型。

  • 返回值：无。

Set (p1, p2)

  • 功能：设置线段的起始点p1和终止点p2。

  • 参数：

    • p1：线段的起始点，sc2Vector类型。

    • p2：线段的终止点，sc2Vector类型。

  • 返回值：bool类型，表示是否设置成功。

SetP1(p)

  • 功能：设置线段的起始点。

  • 参数：

    • p：线段的起始点，sc2Vector类型。

  • 返回值：bool类型，表示是否设置成功。

SetP2(p)

  • 功能：设置线段的终止点。

  • 参数：

    • p：线段的终止点，sc2Vector类型。

  • 返回值：bool类型，表示是否设置成功。

GetP1 ()

  • 功能：获取线段的起始点。

  • 参数：无。

  • 返回值：线段的起始点，sc2Vector类型。

GetP2 ()

  • 功能：获取线段的终止点。

  • 参数：无。

  • 返回值：线段的终止点，sc2Vector类型。

GetLine()

  • 功能：获取当前线段所在的直线。

  • 参数：无。

  • 返回值：线段所在的直线，scLine类型。

GetRotation()

  • 功能：获取线段的旋转角度。

  • 参数：无。

  • 返回值：线段的旋转角度，scRadian类型。
