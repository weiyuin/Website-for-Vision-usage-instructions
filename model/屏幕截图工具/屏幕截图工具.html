屏幕截图工具

功能

对某一个新增的视图View进行截屏，并保存成jpg或者bmp格式的图片。

参数

分类 参数名称 参数描述
属性窗口 截屏窗口 选择需要截屏的窗口，比如，View-1等。注意，如果没有View视图，需要新增一个视图，将要截屏保存图像的拖进View里面。
自动命名 是否使用系统给截图命的名，选择否的话，就需要在数据链中给出文件名，如果文件名为空则按自动命名保存。
覆盖保存 自动名选择“否”，文件名参数不为空时，是否覆盖同名文件保存。
文件格式 有自动，BMP和JPG三种，其中自动和BMP都是保存成BMP格式。JPG格式需要设置压缩比例和压缩质量。
缩放比例 对保存后图片整体大小的缩放。
压缩比例 只有JPG图像格式需要设置压缩比例。
压缩质量 只有JPG图像格式需要设置压缩质量。压缩质量最好是80以上，不会影响图像的清晰度。
是否获取子图 截图是否用子图。如果不是全屏的情况下，图像的左上角为坐标原点；如果是全屏的模式下，View窗口的左上角为坐标原点。
X 子图的起始X坐标。
Y 子图的起始Y坐标
W 子图的宽度。
H 子图的高度。
输出文件路径 截图输出的文件路径。
是否保存图像 控制截图图像是否保存到指定路径。
图像窗口 无 无
数据链 启用 是否启用截图，默认启用。
文件名 截图之后保存的图像文件名称。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 无 无
