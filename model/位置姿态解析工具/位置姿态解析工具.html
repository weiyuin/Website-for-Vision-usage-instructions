位置姿态解析工具

功能

本工具可将GcCoordPos类型的位置姿态数据转换为sc2Vector型的位置坐标数据与double型旋转角度。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 位置姿态 GcCoordPos型的位置姿态数据结果
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 位置坐标 sc2Vector型位置坐标数据
旋转角度 double型的旋转量
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无 无
数据链 位置坐标 sc2Vector型位置坐标数据，后序工具使用，同监视窗口
旋转角度 double型的旋转量，后序工具使用，同监视窗口

详细说明

无

注意

  • 该工具为位置姿态生成工具的反变换工具，可根据项目需要运用。
