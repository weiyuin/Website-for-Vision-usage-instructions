偏差量计算工具

功能

偏差量计算工具用来计算图像中一点与选定参考点之间的偏差。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无
数据链 图像坐标 图像中一点的图像坐标，数据类型为GcCoordPos（位置姿态）。
参考图像坐标 选定的参考点的图像坐标，数据类型为GcCoordPos（位置姿态）。
标定结果 标定计算工具的输出结果，是图像、平台及轴坐标之间的关系。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 偏差量 图像上一点与选定参考点之间的偏差。
执行结果 工具执行结果。
执行时间 工具执行时间。
属性窗口 无 无
数据链 偏差量 图像上一点与选定参考点之间的偏差，供后序工具使用，同监视窗口的偏差量参数。

详细说明

无

注意

  • 数据链窗口的参数图像坐标和参考图像坐标的数据类型是GcCoordPos（位置姿态），而非scVector（二维向量）。使用前应使用位置姿态生成、数组转位置姿态、轴位置转位置姿态等工具进行数据转换。

  • 数据链窗口的参数标定结果是标定计算工具的标定结果（GcCalibrateResult 型），而非靶标相机标定工具的标定结果（scCalibrateResult型）。
