直线生成工具

功能

直线生成工具是基于两点式、点斜式两种形式进行直线生成操作。

参数

分类 参数名称 参数描述
属性窗口 创建方式 分为点斜式和两点式。
图像窗口 无 无
数据链 线段端点1 输入形成直线的点坐标。
线段端点2 创建方式为“两点式”时，输入形成直线的另一点坐标。
线段角度 创建方式为“点斜式”时，输入形成直线的角度，double类型。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 直线结果 输出生成直线的坐标点、直线的方向向量和旋转角度（与x轴正向的夹角）。
线段结果 创建方式为“两点式”时，输出生成线段的两点坐标。
生成的线段角度 创建方式为“两点式”时，输出线段的角度。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 直线结果 输出生成的直线，供后序工具使用，同监视窗口的直线结果参数。
线段结果 创建方式为“两点式”时，输出生成的线段，供后序工具使用，同监视窗口的线段结果参数。
生成的线段角度 创建方式为“两点式”时，输出生成的线段角度，供后序工具使用，同监视窗口生成的线段角度参数。

详细说明

无

注意

  • 直线生成工具显示生成的直线或线段时，需要将显示结果加到有图片显示的View视图中。
