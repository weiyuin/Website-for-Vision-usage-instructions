点斜式直线生成

功能

本工具用于计算由单个GcCoordPos类型的平台坐标决定的scLine型直线结果。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 平台坐标 单个GcCoordPos类型的平台坐标
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出直线 scLine型直线结果数据
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无 无
数据链 输出直线 scLine型直线结果数据，供后序工具使用，同监视窗口

详细说明

无

注意

  • 点斜式直线生成是由一点坐标加一个旋转量来构建直线。由于GcCoordPos型平台坐标是三个double型的X、Y、D坐标组成的结构体。所以点斜式直线生成工具所用点坐标为X、Y组成的二维点坐标，而直线所需的旋转量为平台坐标的D值。

  • 直线生成工具的应用场景主要是将平台坐标练成直线，从而为多点线对位计算工具的目标/对象直线向量输入。
