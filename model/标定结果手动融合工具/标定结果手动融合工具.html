标定结果手动融合工具

功能

将修改后的标定结果的参数，例如图像坐标，平台坐标，进行组合，生成一个整体的标定结果，供其他工具使用，例如“图像到平台坐标转换工具”的数据链输入。

参数

分类 参数名称 参数描述
属性窗口 旧标定类型 标识当前要链接旧标定结果（GcCalibrateResult）还是新标定结果（GeneralCalibResult）
图像窗口 无 无
数据链 图像坐标 标定结果的图像坐标，通常是标定时用来定位的Mark点的图像坐标, GcCoordPos类型。
平台坐标 标定结果的平台坐标，通常是修改后的平台坐标，GcCoordPos类型。
输入标定结果 被修改的原始标定结果（旧标定结果）
运动标定结果 被修改的原始标定结果（新标定结果）
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 标定结果 修改后的标定结果（旧标定结果）。
输出标定结果 修改后的标定结果（新标定结果）。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 标定结果 修改后的标定结果，供其他工具使用，同监视窗口参数。

详细说明

无

注意

  • 数据链中的“图像坐标”、“平台坐标”可以使用“位置姿态生成工具”来获得。
