数组转轴位置

功能

本工具可将三个double型的数据组成的数组（通常为三元数组生成工具生成的结果）转换为GcPlatformAxisPos类型的轴位置数据。该轴位置结果通常作为标定轴位置获取工具的初始轴位置数据链输入。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 坐标数组 三个double型数据组成的数组，通常由三元数组生成工具生成。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 轴位置 GcPlatformAxisPos类型的轴位置结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 轴位置 输出GcPlatformAxisPos类型轴位置，供后序工具使用，同监视窗口参数。

详细说明

无

注意

  • 该类型转换工具应用场景不多，根据项目当需要通过3个double数据数组构建轴位置时，可使用此工具。
