对数运算

功能

对输入数据进行指定运算类型的对数计算。

参数

分类 参数名称 参数描述
属性窗口 输入数据类型 待计算的数据类型，支持double、int两种。
运算类型 对数运算类型，分Ln、Log2、Log10三种。
图像窗口 无 无
数据链 输入数据 待计算的数据，其类型由属性窗口中的“输入数据类型”决定，要求数值必须大于0。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 计算结果 计算的对数运算结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 计算结果 计算的对数运算结果，供后续工具使用。
