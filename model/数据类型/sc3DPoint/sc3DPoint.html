sc3DPoint

功能

表示三维空间点，如圆孔检测工具的输出参数圆孔圆心，点云位置和尺寸测量工具的输出参数特征点。

接口

分类 接口名称 接口描述
构造 sc3DPoint sc3DPoint()默认构造函数，XYZ初始化为0.0
sc3DPoint sc3DPoint (vect)构造函数
sc3DPoint sc3DPoint (x, y, z)构造函数
函数 GetX 获取X坐标
GetY 获取Y坐标
GetZ 获取Z坐标
SetX 设置X坐标
SetY 设置Y坐标
SetZ 设置Z坐标
Map 对当前点进行三维变换得到新的点
a + b sc3DPoint类型的加法，a和b都是sc3DPoint类型的对象
a - b sc3DPoint类型的减法，a和b都是sc3DPoint类型的对象

详细说明

sc3DPoint()

  • 功能：构造sc3DPoint对象，坐标初始化为0.0。

  • 参数：无

  • 返回值：无

sc3DPoint( vect )

  • 功能：使用三维向量构造sc3DPoint对象。

  • 参数：

    • vect：三维向量，sc3Vector类型。

  • 返回值：无

sc3DPoint( x, y, z )

  • 功能：构造sc3DPoint对象。

  • 参数：

    • x：X坐标，浮点数类型。

    • y：Y坐标，浮点数类型。

    • z：Z坐标，浮点数类型。

  • 返回值：无

GetX()

  • 功能：获取X坐标。

  • 参数：无

  • 返回值：浮点数类型

GetY()

  • 功能：获取Y坐标。

  • 参数：无

  • 返回值：浮点数类型

GetZ()

  • 功能：获取Z坐标。

  • 参数：无

  • 返回值：浮点数类型

SetX( x )

  • 功能：设置X坐标。

  • 参数：

    • x：新的X坐标，浮点数类型。

  • 返回值：bool类型，表示是否设置成功

SetY( y )

  • 功能：设置Y坐标。

  • 参数：

    • y：新的Y坐标，浮点数类型。

  • 返回值：bool类型，表示是否设置成功

SetZ( z )

  • 功能：设置Z坐标。

  • 参数：

    • z：新的Z坐标，浮点数类型。

  • 返回值：bool类型，表示是否设置成功

Map( form )

  • 功能：对当前点进行三维变换得到新的点。

  • 参数：

    • form：三维线型变换，sc3Xform类型。

  • 返回值：sc3DPoint类型，表示变换后的新的坐标点

a + b

  • 功能：sc3DPoint类型的加法。如：c = a + b

  • 参数：

    • a：sc3DPoint类型对象

    • b：sc3DPoint类型对象

  • 返回值：sc3DPoint类型

a - b

  • 功能：sc3DPoint类型的减法。如：c = a - b

  • 参数：

    • a：sc3DPoint类型对象

    • b：sc3DPoint类型对象

  • 返回值：sc3DPoint类型
