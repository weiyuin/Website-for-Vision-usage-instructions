GeRotateType

功能

表示旋转镜像工具的旋转类型，是枚举类型。

接口

分类 接口名称 接口描述
属性 eRotate0 无旋转。
eRotate90 旋转90度。
eRotate180 旋转180度。
eRotate270 旋转270度。

详细说明

eRotate0

  • 功能：无旋转。

eRotate90

  • 功能：旋转90度。

eRotate180

  • 功能：旋转180度。

eRotate270

  • 功能：旋转270度。

注意

无。

示例代码

示例1

python mir = GvTool.GetToolData("旋转镜像工具_001.镜像类型") rot = GvTool.GetToolData("旋转镜像工具_001.旋转类型") GvTool.SetToolData("旋转镜像工具_002.镜像类型", mir) GvTool.SetToolData("旋转镜像工具_002.旋转类型", rot)

示例2

python mir = GvVisionAssembly.GeMirrorType ## 无圆括号 rot = GvVisionAssembly.GeRotateType ## 无圆括号 GvTool.SetToolData("旋转镜像工具_002.镜像类型", mir.eNoMirror) GvTool.SetToolData("旋转镜像工具_002.旋转类型", rot.eRotate0)
