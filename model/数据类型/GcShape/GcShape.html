GcShape

功能

表示图形。

接口

分类 接口名称 接口描述
函数 GetShape 获取图形对象。
GetShapeType 获取图形对象的类型。
SetRect 设置基础图形 矩形。
SetAffineRect 设置基础图形 仿射矩形。
SetCircle 设置基础图形 圆形。
SetAnnulusSection 设置基础图形 圆环段。
SetEllipse 设置基础图形 椭圆形。
SetPolyline 设置基础图形 多边形。

详细说明

GetShape()

  • 功能：获取图形对象。

  • 参数：无

  • 返回值：图形对象，数据类型为：scRect、scAffineRect、scCircle、scAnnulusSection、scEllipse、scPolyline之一。

GetShapeType()

  • 功能：获取图形对象的类型。

  • 参数：无

  • 返回值：图形对象的类型，数据类型为ShapeType。

SetRect()

  • 功能：设置基础图形 矩形。

  • 参数：scRect

  • 返回值：bool类型，表示是否设置成功。

SetAffineRect()

  • 功能：设置基础图形 仿射矩形。

  • 参数：scAffineRect

  • 返回值：bool类型，表示是否设置成功。

SetCircle()

  • 功能：设置基础图形 圆形。

  • 参数：scCircle

  • 返回值：bool类型，表示是否设置成功。

SetAnnulusSection()

  • 功能：设置基础图形 圆环段。

  • 参数：scAnnulusSection

  • 返回值：bool类型，表示是否设置成功。

SetEllipse()

  • 功能：设置基础图形 椭圆形。

  • 参数：scEllipse

  • 返回值：bool类型，表示是否设置成功。

SetPolyline()

  • 功能：设置基础图形 多边形。

  • 参数：scPolyline

  • 返回值：bool类型，表示是否设置成功。

注意

  1. 该类型的C++数据类型为std::shared_ptr\<scShape>，主要应用于平面拟合工具2、高度测量工具的ROI参数，在使用时一定要通过GetShape函数获取具体的图形对象。

  2. ROI的类型分为7种：全图、矩形、仿射矩形、圆形、圆环段、椭圆、多边形，GetShapeType函数的返回结果分别为eFull、eRect、eAffineRect、eCircle、eAnnulusSection、eEllipse、ePolyline。eFull时为空指针，使用时应注意。

示例代码

  • 示例1

```python

获取平面拟合工具2_002的ROI区域（ROI区域是一个数组）

Roi=GvTool.GetToolData(“平面拟合工具2_002.ROI”)

获取第一个ROI区域（矩形区域，数据类型为scRect）

shape=Roi[0].GetShape()

获取矩形区域的左上角点（数据类型为sc2Vector）

origin=shape.GetOrigin()

获取矩形区域的左上角点的X坐标（数据类型为浮点型）

origin_x=origin.GetX()

获取矩形区域的左上角点的Y坐标（数据类型为浮点型）

origin_y=origin.GetY() ```

  • 示例2

```python

初始化图形数组和图形

vctShape = GvVisionAssembly.GcShapeVec()

设置第一个图形 矩形

shape1 = GvVisionAssembly.GcShape() rect =GvVisionAssembly.scRect(GvVisionAssembly.sc2Vector(100,100),GvVisionAssembly.sc2Vector(200,300)) shape1.SetRect(rect) vctShape.append(shape1)

设置第二个图形 圆形

shape2 = GvVisionAssembly.GcShape() circle = GvVisionAssembly.scCircle(GvVisionAssembly.sc2Vector(100, 100), 50) shape2.SetCircle(circle) vctShape.append(shape2)

使用脚本构建的ROI设置平面拟合工具2的ROI参数

vct=GvTool.SetToolData(“平面拟合工具2_002.ROI”,vctShape) ```
