GeMirrorType

功能

表示旋转镜像工具的镜像类型，是枚举类型。

接口

分类 接口名称 接口描述
属性 eNoMirror 无旋转镜像。
eHorMirror 水平镜像。
eVerMirror 垂直镜像。
eHorVerMirror 水平垂直镜像。

详细说明

eNoMirror

  • 功能：无镜像。

eHorMirror

  • 功能：水平镜像。

eVerMirror

  • 功能：垂直镜像。

eHorVerMirror

  • 功能：水平垂直镜像。

注意

无

示例代码

示例1

```python

mir = GvTool.GetToolData(“旋转镜像工具_001.镜像类型”)

rot = GvTool.GetToolData(“旋转镜像工具_001.旋转类型”)

GvTool.SetToolData(“旋转镜像工具_002.镜像类型”, mir)

GvTool.SetToolData(“旋转镜像工具_002.旋转类型”, rot)

```

示例2

```python

mir = GvVisionAssembly.GeMirrorType ## 无圆括号

rot = GvVisionAssembly.GeRotateType ## 无圆括号

GvTool.SetToolData(“旋转镜像工具_002.镜像类型”, mir.eNoMirror)

GvTool.SetToolData(“旋转镜像工具_002.旋转类型”, rot.eRotate0)

```
