scImageRGB

功能

表示彩色图像。

接口

分类 接口名称 接口描述
构造 scImageRGB scImageRGB ()默认构造函数
函数 Width 获取图像宽度
Height 获取图像高度
GetPixelValue 获取指定位置的像素值
SetPixelValue 设置指定位置的像素值
GetSubImage 获取子图像

详细说明

scImageRGB()

  • 功能：默认构造函数。

  • 参数：无

  • 返回值：无

Width()

  • 功能：获取图像的宽度。

  • 返回值：图像宽度，整型。

Height()

  • 功能：获取图像的高度。

  • 返回值：图像高度，整型。

GetPixelValue(x, y)

  • 功能：获取(x, y)位置像素各个通道的值，返回结果为元组(r, g, b)，一定要注意图像大小，传入的x，y不能越界。

  • 参数：

    • x：图像坐标系x坐标，整型。

    • y：图像坐标系y坐标，整型。

  • 返回值：各个通道的值组合成的元组(r, g, b)，tuple类型。

    • r：获取的r通道像素值，范围(0\~255)，整型。

    • g：获取的g通道像素值，范围(0\~255)，整型。

    • b：获取的b通道像素值，范围(0\~255)，整型。

SetPixelValue(x, y, r, g, b)

  • 功能：设置(x, y)位置像素的rgb值，一定要注意图像大小，传入的x，y不能越界。

  • 参数：

    • x：图像坐标系x坐标，整型。

    • y：图像坐标系y坐标，整型。

    • r：r通道像素值，范围(0\~255)，整型。

    • g：g通道像素值，范围(0\~255)，整型。

    • b：b通道像素值，范围(0\~255)，整型。

  • 返回值：无。

GetSubImage(img, rect)

  • 功能：获取ROI区域为rect的子图像。

  • 参数：

    • img：获取的子图像，scImageRGB类型。

    • rect：提取子图像的ROI区域，scRect类型。

  • 返回值：整型。

    • 0：执行成功。

    • 1025：图像内存为空或图像大小不合法。

    • 1541：内存分配失败。

    • 2049：无效的区域。

注意

无。

示例代码

```python raw_img = GvTool.GetToolData(“彩色图像文件_004.输出图像RGB”)

获取像素值

r, g, b = raw_img.GetPixelValue(1255, 575) print(r, g, b)

设置像素值

raw_img.SetPixelValue(10, 10, 255, 255, 255)

获取子图像

sub_img = GvVisionAssembly.scImageRGB() roi_rect = GvVisionAssembly.scRect(0, 0, 100, 100) status = raw_img.GetSubImage(sub_img, roi_rect) print(status)

GvTool.SetToolData(“彩色图像转灰度图像工具_006.输入图像”,sub_img) ```
