sc3DPlane

功能

表示三维平面。

详细说明

sc3DPlane()

  • 功能：构造sc3DPlane对象， 默认构造函数。

  • 参数：无

  • 返回值：无

sc3DPlane(vect)

功能：根据三维向量vect构造平面，该平面必经过原点（0，0，0）且法向量为vect，若三维向量vect为零向量，则修正为默认向量(0, 0, 1)。

  • 参数：

    • vect：三维向量，sc3Vector类型。

  • 返回值：无

sc3DPlane(pos, nor)

功能：根据平面所经过点的位置坐标pos和法向量nor构造平面，若法向量nor为零向量，则修正为默认向量(0, 0, 1)。

  • 参数：

    • pos：位置坐标，sc3Dpoint类型。

    • nor：法向量，sc3Dpoint类型。

  • 返回值：无

sc3DPlane(pt1, pt2,pt3)

功能：用三个点pt1、pt2、pt3来构造平面，若三点共线，则返回经过点pt1，且平行于XOY平面的平面。

  • 参数：

    • pt1：平面上第一个点，sc3Dpoint类型。

    • pt2：平面上第二个点，sc3Dpoint类型。

    • pt3：平面上第三个点，sc3Dpoint类型。

  • 返回值：无

sc3DPlane(a, b, c, d)

功能：用一般平面方程 aX + bY + cZ +d = 0构造平面，若系数a、b、c同时为0，则修正为：a = 0, b = 0, c = 1。

  • 参数：

    • a：一般平面方程第一个系数，浮点数类型。

    • b：一般平面方程第二个系数，浮点数类型。

    • c：一般平面方程第三个系数，浮点数类型。

    • d：一般平面方程第四个系数，浮点数类型。

  • 返回值：无

Set3DVector(vect)

功能：设置三维向量vect构造平面，该平面必经过原点（0，0，0）且法向量为vect。

  • 参数：

    • vect：三维向量，sc3Vector类型。

  • 返回值：bool类型，若三维向量vect为零向量 则返回false，否则返回true。

Set3Points(pt1, pt2, pt3)

  • 功能：用平面上三个点来构造平面。

  • 参数：

    • pt1：平面经过的第一个点，sc3Dpoint类型。

    • pt2：平面经过的第二个点，sc3Dpoint类型。

    • pt3：平面经过的第三个点，sc3Dpoint类型。

  • 返回值：bool类型，若三点共线，则返回false，否则返回true。

Get3Points(pt1, pt2,pt3)

  • 功能：获取平面上的三个点。

  • 参数：

    • pt1：平面经过的第一个点，sc3Dpoint类型。

    • pt2：平面经过的第二个点，sc3Dpoint类型。

    • pt3：平面经过的第三个点，sc3Dpoint类型。

  • 返回值：无

SetCoefficients(a, b, c, d)

  • 功能：用平面上三个点来构造平面。

  • 参数：

    • a：一般平面方程第一个系数，浮点数类型。

    • b：一般平面方程第二个系数，浮点数类型。

    • c：一般平面方程第三个系数，浮点数类型。

    • d：一般平面方程第四个系数，浮点数类型。

  • 返回值：bool类型，若系数a，b，c同时为0，则返回false，否则返回true。

GetCoefficients( )

  • 功能：获取平面一般方程 aX + bY + cZ +d = 0 的系数。

  • 参数：无

  • 返回值：各个系数组合成的元组(a, b, c, d)，tuple类型。

SetPos(pos)

  • 功能：设置平面经过的点位置坐标。
  • 参数：
  • pos：位置坐标，sc3Dpoint类型。
  • 返回值：bool类型，表示是否设置成功

GetPos()

  • 功能：获取平面经过的点位置坐标。

  • 参数：无

  • 返回值：平面经过的点位置坐标，sc3Dpoint类型。

SetNormal(nor)

  • 功能：设置平面的法向量。

  • 参数：

    • nor：法向量，sc3Dpoint类型。

  • 返回值：bool类型，若法向量nor为零向量，则返回false，否则返回true。

GetNormal()

  • 功能：获取平面的法向量。

  • 参数：无

  • 返回值：平面的法向量，sc3Dpoint类型。

GetRegularizationNormal()

  • 功能：获取单位化的平面法向量。

  • 参数：无

  • 返回值：sc3Dpoint类型

SetOffset(offset)

  • 功能：设置平面的偏移量。

  • 参数：

    • offset：偏移量，浮点数类型。

  • 返回值：bool类型，表示是否设置成功。

GetOffset()

  • 功能：获取平面的偏移量。

  • 参数：无

  • 返回值：平面偏移量，浮点数类型。

SetRotation(rot)

  • 功能：设置平面法向量的旋转角度。

  • 参数：

    • rot：旋转角度，取值范围[0, π]，scRadian类型。

  • 返回值：bool类型，表示是否设置成功

GetRotation()

  • 功能：获取平面法向量的旋转角度。

  • 参数：无

  • 返回值：scRadian类型

SetTilt(til)

  • 功能：设置平面法向量的倾斜角度。

  • 参数：

    • til：旋转角度，取值范围[0, 2π)，scRadian类型。

  • 返回值：bool类型，表示是否设置成功

GetTilt()

  • 功能：获取平面法向量的倾斜角度。

  • 参数：无

  • 返回值：scRadian类型

Distance(pos)

  • 功能：计算给定点pos到当前平面的距离

  • 参数：

    • pos：给定点，sc3Dpoint类型。

  • 返回值：pos到当前平面的距离，浮点数类型。
