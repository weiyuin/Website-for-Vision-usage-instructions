scImageGray

功能

表示灰度图像。

接口

分类 接口名称 接口描述
构造 scImageGray scImageGray ()默认构造函数
函数 Width 获取图像宽度
Height 获取图像高度
GetPixelValue 获取指定位置的像素值
SetPixelValue 设置指定位置的像素值
GetSubImage 获取子图像

详细说明

scImageGray()

  • 功能：默认构造函数。

  • 参数：无

  • 返回值：无

Width()

  • 功能：获取图像的宽度。

  • 返回值：图像宽度，整型。

Height()

  • 功能：获取图像的高度。

  • 返回值：图像高度，整型。

GetPixelValue(x, y)

  • 功能：获取(x, y)位置像素灰度值，一定要注意图像大小，传入的x，y不能越界。

  • 参数：

    • x：图像坐标系x坐标，整型。

    • y：图像坐标系y坐标，整型。

  • 返回值：(x, y)坐标位置的像素灰度值，double类型。

SetPixelValue(x, y, value)

  • 功能：设置(x, y)位置像素灰度值，一定要注意图像大小，传入的x，y不能越界。

  • 参数：

    • x：图像坐标系x坐标，整型。

    • y：图像坐标系y坐标，整型。

    • value：像素灰度值。

  • 返回值：无。

GetSubImage(img, rect)

  • 功能：获取ROI区域为rect的子图像。

  • 参数：

    • img：获取的子图像，scImageGray类型。

    • rect：提取子图像的ROI区域，scRect类型。

  • 返回值：整型。

    • 0：执行成功。

    • 1025：图像内存为空或图像大小不合法。

    • 1541：内存分配失败。

    • 2049：无效的区域。

注意

无

示例代码

python raw_img = GvTool.GetToolData("图像文件_001.输出图像") sub_img = GvVisionAssembly.scImageGray() roi_rect = GvVisionAssembly.scRect(0, 0, 100, 100) status = raw_img.GetSubImage(sub_img, roi_rect)
