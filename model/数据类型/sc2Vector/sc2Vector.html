sc2Vector

功能

表示二维向量，例如几何定位工具的输出参数搜索坐标，找圆工具的输出参数圆心，二维向量生成工具的输出，位置姿态生成工具的输入参数位置坐标都是该类型。

接口

分类 接口名称 接口描述
构造 sc2Vector sc2Vector()默认构造函数，XY初始化为0.0
sc2Vector sc2Vector (x, y)构造函数
函数 GetX 获取X坐标
GetY 获取Y坐标
SetX 设置X坐标
SetY 设置Y坐标
a + b sc2Vector类型的加法，a和b都是sc2Vector类型的对象
a - b sc2Vector类型的减法，a和b都是sc2Vector类型的对象

详细说明

sc2Vector( )

  • 功能：构造sc2Vector对象，坐标初始化为0.0。

  • 参数：无

  • 返回值：无

sc2Vector(x, y)

  • 功能：构造sc2Vector对象。

  • 参数：

    • x：X坐标，浮点数类型。

    • y：Y坐标，浮点数类型。

  • 返回值：无

GetX( )

  • 功能：获取X坐标。

  • 参数：无

  • 返回值：浮点数类型。

GetY( )

  • 功能：获取Y坐标。

  • 参数：无

  • 返回值：浮点数类型。

SetX(newX )

  • 功能：设置X坐标。

  • 参数：

    • newX：新的X坐标，浮点数类型。

  • 返回值：bool类型，表示是否设置成功。

SetY(newY )

  • 功能：设置Y坐标。

  • 参数：

    • newY：新的Y坐标，浮点数类型。

  • 返回值：bool类型，表示是否设置成功。

a + b

  • 功能：sc2Vector类型的加法。如：c = a + b

  • 参数：

    • a：sc2Vector类型对象。

    • b：sc2Vector类型对象。

  • 返回值：sc2Vector类型。

a - b

  • 功能：sc2Vector类型的减法。如：c = a – b。

  • 参数：

    • a：sc2Vector类型对象。

    • b：sc2Vector类型对象。

  • 返回值：sc2Vector类型。
