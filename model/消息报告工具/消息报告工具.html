消息报告工具

功能

该工具主要用于在窗口输出提示信息，可以设置消息报告类型，来改变消息的提示级别。

参数

分类 参数名称 参数描述
属性窗口 报告类型 有提示/警告/错误/致命错误/弹出提示/选项提示/自定义提示几个类型，灵活组合能够很好的获取程序运行错误信息。
窗口尺寸 报告类型选择“自定义提示”，消息提示窗口的大小，分大、中、小三种尺寸。
窗口位置 报告类型选择“自定义提示”，消息提示窗口的显示位置，居中、左上角、左下角、右上角、右下角。
按钮个数 报告类型选择“自定义提示”，消息提示窗口包含的按钮个数，最大值为6。
窗口标题 报告类型选择“选项提示/自定义提示”，消息提示窗口标题，标题内容限制中文不超过8个字符，英文不超过16个字符。
窗口风格 报告类型选择“选项提示/自定义提示”，消息提示的风格，包括三种：默认风格、红色风格、黄色风格。
按钮名称 报告类型选择“自定义提示”，消息提示窗口中按钮的名称，按钮名称限制中文不超过8个字符，英文不超过16个字符。
图像窗口 无 无
数据链 消息字符串 需要提示的消息，消息长度限制中文不超过64个字符，英文不超过128个字符。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 所选按钮 报告类型选择“自定义提示”，用户点击的按钮序号，从1开始，若未点击按钮，则结果为-1。
执行结果 工具执行结果。
执行时间 工具执行时间。
消息字符串 提示的消息。
图像窗口 无 无
数据链 无 无
