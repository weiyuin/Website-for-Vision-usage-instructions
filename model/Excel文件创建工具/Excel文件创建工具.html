Excel文件创建工具

功能

EXCEL文件创建工具主要功能是将数据存储到Excel文件中保存。

参数

分类 参数名称 参数描述
属性窗口 自动命名 是否自动为excel文件命名（是-以序号+时间的格式自动命名；否-须在参数链中设置文件名）。
输出文件路径 文件存储路径。
图像窗口 无
数据链 EXCEL文件名 属性窗口-自动命名选择为否时设置（命名格式：*.csv）。
EXCEL表头数据 EXCEL文件表头,若要创建多个表头，则各个表头用逗号隔开（Head1，Head2…）。
EXCEL数据 对应表头下的数据（对应表头格式，各列数据用逗号隔开）。
是否换行 每次执行写入新数据时，是否换行写入。
高级界面 无

输出

分类 参数名称 参数描述
输出项 执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无
数据链 无

详细说明

无

注意

  • 由于各个表头名需要逗号隔开，因此创建多个表头时，可与数据包生成工具和字符串解析工具来管理、组合表头名。同时的，对应不同的表头，其数据也可以用相同的方式来组合。
