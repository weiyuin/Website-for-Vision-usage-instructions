振镜打标位置计算工具

功能

振镜打标位置计算工具是将输入的图像坐标经过计算后，得到转换后的振镜坐标。

参数

分类 参数名称 参数描述
属性窗口 是否忽略角度变化 控制转换计算过程中是否忽略转换图像坐标的角度变化，选择“是”，计算过程中角度值设为0。
是否采用多基准 选择“是”，转换计算过程中，利用距离转换图像坐标最近的辅助点来计算对应的振镜坐标；选择“否”，利用九圆标定区域中的中间点(第5个辅助点)来计算对应的振镜坐标。
是否采用绝对角度 采用绝对角度指将输出振镜坐标的旋转角度规范化到[-PI/2,PI/2]。
输入角度归一化 将输入的角度归一化，取值为[-90°,90°)、[-180°,180°)、[0°,180°)、[0°,360°)；比如归一化到-90~90°之间，输入的是92°，内部计算的时候会变成-88°。
图像窗口 无 无
数据链 图像坐标 输入需要转换的图像坐标。
产品上料角度 链接能够标识产品角度的数据，通常为产品上某个特征的角度。
X轴方向 振镜X坐标轴方向。
Y轴方向 振镜Y坐标轴方向。
九圆标定结果 输入图像坐标与振镜坐标关联后图像坐标点集，从九圆关联标定工具链接得到。
X补偿值 对计算结果中的X值做补偿。
Y补偿值 对计算结果中的Y值做补偿。
D补偿值 对计算结果中的D值做补偿。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 振镜坐标X 输出转换后的振镜坐标X。
振镜坐标Y 输出转换后的振镜坐标Y。
振镜坐标D 输出转换后的振镜坐标D。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 振镜坐标X 输出转换后的振镜坐标X，同监视窗口。
振镜坐标Y 输出转换后的振镜坐标Y，同监视窗口。
振镜坐标D 输出转换后的振镜坐标D，同监视窗口。
