中点生成工具

功能

中点生成工具可以用来获得图像上两个点的中点，并输出该点的图像坐标，供后序工具使用。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 输入点1 输入第一个点的图像坐标。
输入点2 输入第二个点的图像坐标。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出中点 输出输入点1和输入点2的中点图像坐标。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出中点 输出输入点1和输入点2的中点图像坐标，供后序工具使用，同监视窗口的输出中点参数。

详细说明

无

注意

  • 中点生成工具的参数来源一般为交点工具输出的交点。
