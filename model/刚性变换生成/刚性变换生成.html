刚性变换生成工具

功能

该工具主要针对实际工程中，当目标图像无法提供很好的定位信息输出二维线性变换结果，或者传统的定位方式无法对指定对象提供二维线性变换时，该工具能够利用找线、卡尺等工具灵活的将目标的旋转平移信息生成二维线性变换供后续工具使用。

参数

分类 参数名称 参数描述
属性窗口 重置参考位置 此参数需要特别注意设置的方法。
选择是：则获取新的参考位置（一般来说，需要设置参考位置时才将此参数设置为是。特别的，如果参考位置一直有独立的数据源，才可以在实时检测时将此参数设置为否）。
选择否：实时检测时需要将该参数设置为否，此时将使用之前的参考数据（第一次设置参考位置时，需先将该参数设置为是，以获取训练的参考位置参数）。
是否耦合角度 否：平移单独取自参考点，是：平移取自参考点平移和旋转角度
图像窗口 无 无
数据链 位置参考值 参考位置的坐标，用于计算实时位置的改变（参考位置获取方式跟参数选择重置参考有关），scVector类型。
实时位置 实时位置的坐标，用于计算实时位置的改变，scVector类型。
角度参考值 参考位置的角度，用于计算实时角度变换（参考位置获取方式跟参数选择重置参考有关），double类型。
实时角度 实时位置的角度，用于计算实时角度变换，double类型。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 二维线性变换 实时位置相对于参考位置的平移、旋转、缩放变换
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 二维线性变换 输出二维线性变换，供其他工具使用，同监视窗口参数。
