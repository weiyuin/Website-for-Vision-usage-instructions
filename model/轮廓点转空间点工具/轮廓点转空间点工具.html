轮廓点转空间点工具

功能

根据输入的深度图像、轮廓图像以及向量ROI，计算轮廓点在深度图上的对应点坐标。即将二维坐标转为三维空间点，用于后续的点云测量等点云量测类工具中。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 输入深度图像 输入图像宽度、高度、像素大小，同图像窗口的输入图像参数。
输入轮廓图像 输入待检测的轮廓图像的轮廓数据，该参数来源于截面生成工具的输出轮廓数据参数。
输入向量ROI 该参数来源于截面生成工具的向量交互ROI属性参数。
输入轮廓点 输入轮廓图像中获取到的轮廓点坐标，该参数来源于轮廓位置测量工具的输出特征点坐标参数。
是否清空数据 每次计算时三维空间点数组结果清空。
高级界面 无 无

输出

分类 参数名称 参数描述（参数的含义）
监视窗口 三维空间点 转换后的三维空间点坐标。
三维空间点数组 转换后的三维空间点坐标数组。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 无 无

详细说明

无

注意

  • “轮廓点转空间点工具”的输出结果“三维空间点”可以使用脚本工具GvTool.GetToolData得到并转出。具体参见“轮廓点转空间点.gvp”例程。
