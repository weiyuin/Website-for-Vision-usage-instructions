基准平台坐标修正工具

功能

基准平台坐标修正工具是用来修改标定结果中平台坐标的，即旋转中心。将原始标定结果中的平台坐标X修改为“平台坐标X修正量”、原始标定结果中的平台坐标Y修改为“平台坐标Y修正量”。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 原始标定结果 待修改的标定结果。
平台坐标X修正量 标定结果中平台坐标X修改之后的值，输入double型，可以通过自己定义变量获得。
平台坐标Y修正量 标定结果中平台坐标Y修改之后的值，输入double型，可以通过自己定义变量获得。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 修正后的标定结果 修改平台坐标之后的标定结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 修正后的标定结果 输出修改了平台坐标之后的标定结果，供其他工具使用，同监视窗口“修正后的标定结果”参数。
