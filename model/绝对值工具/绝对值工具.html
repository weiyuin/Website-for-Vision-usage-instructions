绝对值工具

功能

绝对值工具可以用来求解输入数据的绝对值。

参数

分类 参数名称 参数描述
属性窗口 数据类型 输入数据的数据类型，绝对值工具提供的数据类型分为2种：double和int。
图像窗口 无 无
数据链 输入数据 绝对值工具的输入值。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 绝对值 计算后得到的绝对值。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 绝对值 输出计算后得到的绝对值，供后序工具使用，同监视窗口的绝对值参数。

详细说明

无

注意

  • 数据链窗口输入数据的类型和属性窗口的数据类型必须一致，否则数据链参数找不到来源，绝对值工具不能执行。
