数组转位置姿态

功能

本工具可将三个double型的X、Y、D坐标数据组合成的数组转换为GcCoordPos类型位置姿态数据。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 坐标数组 3个double类型数据的数组。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 位置姿态 GcCoordPos类型的位置姿态数据结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 位置姿态 输出GcCoordPos类型的位置姿态结果，供后序工具使用，同监视窗口参数。

详细说明

无

注意

  • 该工具的输入是三元数组。
