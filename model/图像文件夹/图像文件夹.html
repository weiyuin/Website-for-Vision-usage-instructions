图像文件夹工具

功能

图像文件夹工具就是根据文件路径依次加载文件夹中的图片。

参数

分类 参数名称 参数描述
属性窗口 文件夹路径 需要加载图片的文件夹。
开启过滤功能 开发过滤功能会以过滤条件的正则表达式对文件筛选。
过滤条件 用于过滤的正则表达式。
图像窗口 无 无
数据链 无 无
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出图像 灰度图像输出，包括图像长宽和像素大小。
输出图像RGB 彩色图像输出，包括图像长宽和像素大小。
输出深度图像 深度图像输出，包括图像长宽和像素大小。（可以右键查看3D效果）
输出轮廓图像 轮廓图像输出，包括轮廓总点数和有效点数。
当前文件 当前图片的路径文件名。
当前文件名称 当前图片的名称。
遍历到达末尾 是否遍历到文件末尾。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 输出图像 显示路径加载的图像。
执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 输出图像 输出灰度图像或者彩色图像或者深度图像或者轮廓图像。
输出图像RGB 彩色图像输出，包括图像长宽和像素大小。
遍历到达末尾 文件夹中的文件是否遍历到末尾
