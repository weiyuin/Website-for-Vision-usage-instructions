角度归一化工具

功能

角度归一化工具是对输入的角度进行转换，使之限定在一定的范围内。

参数

分类 参数名称 参数描述）
属性窗口 角度归一化方式 角度归一化是指对输入的角度进行转换，使之限定在一定的范围内。角度归一化有4种，[-90°, 90°)、[-180°, 180°)、[0°, 180°) 和[0°, 360°)。
图像窗口 无 无
数据链 输入角度 输入进行归一化的角度，double类型。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出角度 输出转化后的角度。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出角度 输出转化后的角度，供后序工具使用，同监视窗口的输出角度参数。
