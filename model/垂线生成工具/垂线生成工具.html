垂线生成工具

功能

垂线生成工具是在参考点的位置生成参考直线垂线的操作。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 参考点 输入生成垂线的参考点。
参考直线 输入生成垂线的参考直线。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出直线 输出生成垂线的坐标点、方向向量和旋转角度。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出直线 输出生成的垂线，供后序工具使用，同监视窗口的输出直线参数。

详细说明

无

注意

  • 垂线生成工具显示生成的垂线时，需要将显示结果加到有图片显示的View视图中。
