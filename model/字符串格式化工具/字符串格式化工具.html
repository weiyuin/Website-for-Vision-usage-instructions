字符串格式化工具

功能

对输入的字符串，按照一定的格式要求进行格式化。

参数

分类 参数名称 参数描述
属性窗口 字符串格式 输入指定的格式化字符串。
图像窗口 无 无
数据链 数据 输入格式化的数据。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 格式化结果 输出格式化后的结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 格式化结果 输出格式化后的结果，供后序工具使用，同监视窗口的格式化结果参数。

详细说明

字符串格式

字符串格式输入的指定格式为：%[flags] [width] [.precision] [h | l]type，其中type是唯一必需的字段，其他为可选字段，各字段的说明如表1所示。

表1 指定输入字符串格式

格式 说明
[flags] 控制输出的对齐和输出的符号，十六进制前缀。-指定参数向左对齐，默认是向右对齐。+/-正负号。#与0x、0X一起使用，表示以十六进制格式化。输入#时格式化带有十六进制前缀，反之没有前缀，格式化输出的位数可以设置，范围为0~9，例如%04x，输出为四位。
[width] 指定被格式化的值所占的宽度，输入值得范围0~99。
[.precision] 对于浮点数指定精度，输入范围0~9。
[h | l] 指定相应参数的大小，long、double可以加l，short可以加h。
type 转换说明符，指定转换为字符、字符串、整数或浮点数，是唯一必需的格式字符，可放在任意可选字段之后；float：%f；double：%f/%lf；string：%s；int/ bool：%d； long：%ld； short：%d/%hd；char：%c。

注意

  • 字符串格式化工具没有配置数据参数时，输出为格式化字符串。

  • 字符串配置数据参数时，与数据包生成工具配合使用。
