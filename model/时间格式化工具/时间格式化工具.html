时间格式化工具

功能

对当前系统时间按照一定的格式要求进行格式化。

参数

分类 参数名称 参数描述
属性窗口 时间格式 输入指定的时间格式化字符串。
图像窗口 无 无
数据链 无 无
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 格式化结果 输出格式化后时间结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 格式化结果 输出格式化后时间结果，供后序工具使用，同监视窗口的格式化结果参数。

详细说明

时间格式

时间格式输入的指定格式如表1所示。

表1 指定输入时间格式

%a：星期缩写，例如Thu。 %A：星期全写，例如Thursday。
%b：月份缩写，例如Aug。 %B：月份全写，例如August。
%c：日期和时间表示，例如Thu Aug 23 14:55:02 2001。 %D：月份中的第几天，01-31。
%H：24小时制中的第几个小时，00-23。 %j：一年中的第几天，001-366。
%m：十进制的月份序号，01-12。 %M：分钟，00-59。
%S：秒，00-61。 %U：周编号，以第一个星期日作为第一周的第一天，00-53。
%w：工作日编号，以星期日作为第一天0，0-6。 %W：周编号，以第一个星期一作为第一周的第一天，00-53。
%x：日期表示，月/日/年，例如2017年1月11日表示为01/11/17。 %X：时间表示，例如14:55:02。
%y：年的最后两位数，例如2017年表示为17。 %Y：年的完整表示，例如2017。
%p：上午或下午，AM或PM。 %ms：毫秒的表示，目前只支持在字符串最后添加使用
