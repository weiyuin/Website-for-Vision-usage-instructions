轮廓提取工具结果解析

功能

轮廓提取工具结果解析用于对轮廓提取工具的结果进行解析处理，将轮廓提取工具的其中一个轮廓链，解析为一组轮廓点集。

参数

分类 参数名称 参数描述
属性窗口 是否转为Sc2Vector集合 控制解析后轮廓点链表集合的数据类型，选择“是”为Sc2Vector类型，选择“否”为scFeaturelet类型。
图像窗口 无
数据链 轮廓提取结果 输入经过数组解析工具处理的轮廓数据。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 轮廓点链表集合 是否转为Sc2Vector集合参数选择“否”时，输出解析后的轮廓点集，数据类型为scFeaturelet。
轮廓点集合 是否转为Sc2Vector集合参数选择“是”时，输出解析后的轮廓点集，数据类型为Sc2Vector。
轮廓点总数 输出轮廓点集的个数。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 轮廓点链表集合 输出解析后的轮廓点集，供后序工具使用，同监视窗口的轮廓点链表集合参数。
轮廓点集合 输出解析后的轮廓点集，供后序工具使用，同监视窗口的轮廓点集合参数。
轮廓点总数 输出轮廓点集个数，同监视窗口的轮廓点总数参数。

详细说明

无

注意

  • 轮廓提取工具结果解析与数组解析工具、轮廓提取工具配合使用，其中数组解析工具属性窗口的数据类型参数选择scFeatureletVector。
