三角函数工具

功能

三角函数工具可以用来求解输入数据的三角函数值。

参数

分类 参数名称 参数描述
属性窗口 计算类型 选择某种三角函数类型，计算对应三角函数值。三角函数工具提供的计算类型分为6种：正弦、余弦、正切、反正弦、反余弦和反正切。
图像窗口 无 无
数据链 输入数据 三角函数的输入值（double型）。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出结果 计算后得到的三角函数值。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出结果 计算后得到的三角函数值，供后序工具使用，同监视窗口的输出结果参数。

详细说明

无

注意

  • 三角函数工具的输入数据类型仅支持double类型。
