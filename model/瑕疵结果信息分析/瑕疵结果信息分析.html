瑕疵结果信息分析工具

功能

瑕疵结果信息分析工具用于对瑕疵检测工具的瑕疵结果进行解析处理。由于瑕疵检测工具的瑕疵结果参数类型为DefectResultInfo，需要先使用数组解析工具将瑕疵检测工具的输出瑕疵结果中某一组数据取出来，具体取出的是哪组数据和数据解析工具的输入数据索引有关系，再使用瑕疵结果信息分析工具将该组数据解析成单个结果数据。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 瑕疵结果信息 类型为“DefectResultInfo”的数组解析工具的输出数据作为输入。数组解析工具的输入是瑕疵检测工具的瑕疵结果输出。
高级界面 无 无

外接矩形宽度 对应索引值的瑕疵的坐标最小外接矩形的宽度，瑕疵检测工具的检测类型选择基于面积时，该值为0。

注意

  • 瑕疵结果信息分析工具不能单独使用，需要与数组解析工具配合使用，数组解析工具瑕疵结果解析为一组数据，再使用瑕疵结果信息分析工具将该组数据解析成单个结果数据。

  • 数组解析工具在与瑕疵结果信息分析工具搭配使用时，其属性窗口的数据类型参数应选择为DefectResultInfo，否则在数据链窗口配置参数输入数组时会找不到参数来源。
