直线图像到平台转换

功能

在对位流程中，相机对产品拍照查找得到图像上的直线（线段），不能直接用于对位计算。需要根据标定结果，将图像上的直线（线段）转换到平台坐标系下，才能进行之后的对位偏移量的计算。本工具就是用于完成图像上的直线（线段）到平台坐标系的转换，并输出转换后的平台坐标系坐标。

参数

分类 参数名称 参数描述
属性窗口 转换工具类型 不包含轴位置变化：标定与对位时的轴位置无变化
包含平台轴位置变化：标定与对位时的平台轴位置变化
包含相机轴位置变化：标定与对位时的相机轴位置变化
包含平台与相机轴位置变化：标定与对位时的平台轴和相机轴位置均发生变化
图像窗口 无 无
数据链 输入线段图像坐标 待转换为平台坐标系下的原始图像上的线段
标定结果 标定计算过程输出的标定结果
标定时平台轴位置 标定时的平台轴位置，只在工具类型为“包含平台/平台与相机轴位置变化”时使用
对位时平台轴位置 对位时的平台轴位置，只在工具类型为“包含平台/平台与相机轴位置变化”时使用
平台轴方向 平台各轴的正方向，只在工具类型为“包含平台/平台与相机轴位置变化”时使用
标定时相机轴位置 标定时的相机轴位置，只在工具类型为“包含相机/平台与相机轴位置变化”时使用
对位时相机轴位置 对位时的相机轴位置，只在工具类型为“包含相机/平台与相机轴位置变化”时使用
相机轴方向 相机各轴的正方向，只在工具类型为“包含相机/平台与相机轴位置变化”时使用
相机轴标定结果 相机轴标定过程输出的标定结果，只在工具类型为“包含相机/平台与相机轴位置变化”时使用
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 直线平台坐标 转换后的平台坐标下的直线结果
线段平台坐标结果 转换后的平台坐标下的线段结果
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无
数据链 直线平台坐标 转换后的平台坐标下的直线结果，供后序工具使用，同监视窗口
线段平台坐标结果 转换后的平台坐标下的线段结果，供后序工具使用，同监视窗口

详细说明

转换工具类型

如果没有标定初始轴位置与对位时的轴位置无变化，只有产品随机上料，那么直接从图像坐标转换得到平台坐标，可以直接代入对位计算，即选择“不包含轴位置变化”选项；如果存在轴位置变化，直接从图像坐标转换得到平台坐标，需要根据轴位置变化进行坐标修正，才能代入对位计算，即选择相应的“包含…轴位置变化”选项。
