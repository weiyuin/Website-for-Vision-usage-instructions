光源控制工具

功能

在设置好光源后（主界面-工具条-设置-光源），通过光源控制工具完成对光源开启关闭及亮度的控制。

参数

分类 参数名称 参数描述
属性窗口 光源信息 选择所需要光源设备（在光源理对话框中完成光源配置的光源）。
通道号 在配置光源设备时，可以预设多个通道，每个通道单独的设置光源亮度值，通过选择通道号即根据需求切换光源亮度。
开关 直接控制光源的开启关闭状态。
亮度值 直接改变光源亮度（范围0-255）。
图像窗口 无 无
数据链 无 无
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 无 无

详细说明

无

注意

  • 假设：该工具绑定的光源A有8个通道，该工具会按照对这8个通道的设置，循环的对8个通道进行光源亮度的设置。
