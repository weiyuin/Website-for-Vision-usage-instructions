文件解析导入工具

功能

文件解析导入工具主要目的是解决将工程数据批量导入工程作为输入参数，人工输入过于繁琐的问题。该工具主要通过指定分割字符、数据类型来导入.txt文件。

参数

分类 参数名称 参数描述
属性窗口 数据类型 文件导入数据的类型，int、double、float、string。
分割符号 数据与数据之间的分隔符号，支持分号、逗号和空格。
导入文件 导入文件的路径，支持.txt文件。
数据链 无 无
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 接收的数据 输出从文件导入的数据。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出数据 输出从文件导入的数据。
