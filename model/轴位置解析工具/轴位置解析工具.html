轴位置解析工具

功能

本工具可将GcPlatformAxisPos类型的轴位置数据转换为三个double型的X、Y、D坐标数据。该工具的应用场景是将对位偏移量解析成double型数据以通过通信发送工具发送给机械手。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 平台轴位置 GcPlatformAxisPos型的平台轴位置，可以是对位计算工具的输出。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 轴X位置 double型的轴位置X坐标分量。
轴Y位置 double型的轴位置Y坐标分量。
轴D位置 double型的轴位置D坐标分量。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 轴X位置 double型的轴位置X坐标分量，供后序工具使用，同监视窗口参数。
轴Y位置 double型的轴位置Y坐标分量，供后序工具使用，同监视窗口参数。
轴D位置 double型的轴位置D坐标分量，供后序工具使用，同监视窗口参数。

详细说明

无

注意

  • 该类型转换工具在标定对位工程中配合对位计算工具，将对位计算工具计算出的GcPlatformAxisPos类型的对位偏移量结果解析成double数据，后将double数据通过数据包生成工具打包成数组后通过通信发送工具发送给机械手。
