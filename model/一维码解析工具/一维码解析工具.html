一维码解析工具

功能

解析一维码结果。

属性窗口 无 无
图像窗口 无 无
数据链 读码结果 QR码读码工具的一组数据。
高级界面 无 无

详细说明

无

注意

  • 为了读取一维码读码工具的结果，该工具要与“数组解析工具”联合使用，并且要把“数组解析工具”的数组类型选择为“SingleLinearCodeResult”，作为该工具的输入。
