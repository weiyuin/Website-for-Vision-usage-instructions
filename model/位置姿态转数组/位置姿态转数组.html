位置姿态转数组

功能

本工具可将GcCoordPos型位置姿态数据转换为三个double型的X、Y、D坐标数据组合成的坐标数组。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 位置姿态 GcCoordPos型的位置姿态数据结果
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 坐标数组 三个double型的X、Y、D坐标数据组合成的数组
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无 无
数据链 坐标数组 三个double型的X、Y、D坐标数据组合成的数组

详细说明

无

注意

  • 该工具将GcCoordPos型数据转换到三个double型的X、Y、D坐标数据组合成的坐标数组，可根据项目需要运用。
