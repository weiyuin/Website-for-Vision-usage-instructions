位置姿态拆散

功能

本工具可将GcCoordPos类型的位置姿态数据转换XYD对应的三个double类型数据。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 位置姿态 GcCoordPos型的位置姿态数据结果
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 X值 X坐标，double类型。
Y值 Y坐标，double类型。
D值 D坐标，double类型。
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无 无
数据链 X值 输出X坐标，double类型。供其他工具使用，同监视窗口参数。
Y值 输出Y坐标，double类型。供其他工具使用，同监视窗口参数。
D值 输出D坐标，double类型。供其他工具使用，同监视窗口参数。

详细说明

无

注意

  • 该工具是将GcCoordPos类型的位置姿态数据拆散成三个double类型的坐标数据，而位置姿态解析工具是将GcCoordPos类型的位置姿态数据解析成一个sc2Vector类型的二维向量数据和一个double类型的旋转数据，使用时应注意二者区别。
