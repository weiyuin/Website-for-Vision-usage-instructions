GcScriptGuiArray

功能

用来存储要显示到视图窗口的GUI数据，添加到GcScriptGuiArray中的GUI对象可以显示到视图窗口。

接口

分类 接口名称 接口描述
构造 GcScriptGuiArray GcScriptGuiArray()默认构造函数。
函数 Add(GUI对象) 添加要显示到视图窗口的GUI对象

详细说明

GcScriptGuiArray( )

  • 功能：构造GcScriptGuiArray对象。

  • 参数：无

  • 返回值：无

Add(GUI对象)

  • 功能：将要显示到视图窗口的GUI对象添加到GcScriptGuiArray中。

  • 参数：GUI对象，例如GsScriptGuiCross类型的对象，GsScriptGuiText类型的对象。

  • 返回值：bool类型，表示是否添加成功。

注意

无

示例代码

python guiArray = GvVisionAssembly.GcScriptGuiArray() guiText = GvVisionAssembly.GsScriptGuiText() guiText.sScriptGuiStyle = guiStyle guiText.strText ="text" guiText.posX = 0.0 guiText.posY = 0.0 guiText.deg = 0.0 guiArray.Add(guiText) GvGuiDataAgent.SetGraphicDisplay("View-1", guiArray)
