GsScriptGuiAnnulus

功能

要在视图窗口显示的圆环。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
annulus 要显示的圆环。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

annulus

  • 功能：要显示的圆环，scAnnulus类型

注意

无

示例代码

```python

Gui风格

guistyle = GvVisionAssembly.GsScriptGuiStyle() guistyle.nLineWidth = 2 guistyle.clrLineColor = [0, 255, 0]

圆环Gui对象

pos = GvVisionAssembly.sc2Vector(100, 100) annulus = GvVisionAssembly.scAnnulus(pos, 20, 3) annulus_gui = GvVisionAssembly.GsScriptGuiAnnulus() annulus_gui.sScriptGuiStyle = guistyle annulus_gui.annulus = annulus

Gui数组

guiarray = GvVisionAssembly.GcScriptGuiArray() guiarray.Add(annulus_gui) # 将Gui对象添加到Gui数组

将Gui数组设置到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray) ```
