GsScriptGuiCircle

Chức năng

Hiển thị hình tròn trong cửa sổ xem.

Giao diện

Phân loại Tên giao diện Mô tả giao diện
Thuộc tính sScriptGuiStyle Kiểu hiển thị, bao gồm phông chữ, kích thước, màu sắc, v.v.
circle Hình tròn cần hiển thị.

Giải thích chi tiết

sScriptGuiStyle

  • Chức năng: Kiểu hiển thị, kiểu dữ liệu GsScriptGuiStyle.

circle

  • Chức năng: Hình tròn cần hiển thị, kiểu dữ liệu scCircle.

Lưu ý

Không có.

Ví dụ mã

```python guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.bVisible = True guiStyle.nLineStyle = 1 guiStyle.nLineWidth = 2 guiStyle.clrLineColor = [255, 0, 0] guiStyle.bLabelVisible = True guiStyle.strLabel = “标签” # Nếu không cần hiển thị nhãn label, có thể xóa dòng này guiStyle.strLabelFont = “Arial” guiStyle.lFontSize = 20

guiArray = GvVisionAssembly.GcScriptGuiArray() guiCircle = GvVisionAssembly.GsScriptGuiCircle() center = GvVisionAssembly.sc2Vector(100, 100) # Vị trí tâm hình tròn guiCircle.circle = GvVisionAssembly.scCircle(center, 50) guiCircle.sScriptGuiStyle = guiStyle guiArray.Add(guiCircle)

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
