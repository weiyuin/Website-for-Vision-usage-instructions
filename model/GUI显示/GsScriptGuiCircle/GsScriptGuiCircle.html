GsScriptGuiCircle

功能

要在视图窗口显示的圆形。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
circle 要显示的圆形。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

cicle

  • 功能：要显示的圆形，scCircle类型

注意

无。

示例代码

```python guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.bVisible = True guiStyle.nLineStyle = 1 guiStyle.nLineWidth = 2 guiStyle.clrLineColor = [255, 0, 0] guiStyle.bLabelVisible = True guiStyle.strLabel = “标签” # 如果不需要显示label标签，这一行代码可以删除 guiStyle.strLabelFont = “Arial” guiStyle.lFontSize = 20

guiArray = GvVisionAssembly.GcScriptGuiArray() guiCircle = GvVisionAssembly.GsScriptGuiCircle() center = GvVisionAssembly.sc2Vector(100, 100) # 圆心位置 guiCircle.circle = GvVisionAssembly.scCircle(center, 50) guiCircle.sScriptGuiStyle = guiStyle guiArray.Add(guiCircle)

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
