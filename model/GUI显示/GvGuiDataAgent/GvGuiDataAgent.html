GvGuiDataAgent

功能

用来将GvGuiDataAgent中存储的GUI对象数组显示到视图窗口的GcScriptGuiDataAgent类的对象，禁止自己定义一个GcScriptGuiDataAgent对象。

接口

分类 接口名称 接口描述
函数 SetGraphicDisplay(str, value) 将要显示的GUI对象添加到视图窗口。

详细说明

SetGraphicDisplay(str, value)

  • 功能：将要显示的GUI对象数组添加到视图窗口。

  • 参数：

    • str: str类型，视图名称。例如”View-1”。

    • value: GcScriptGuiArray类型，要添加的GUI对象数组。

  • 返回值：bool类型，表示是否添加成功。

注意

  1. 禁止自己定义一个GcScriptGuiDataAgent对象，内部已经定义了GvGuiDataAgent对象，可以直接使用。

示例代码

1、添加GUI

python guiArray = GvVisionAssembly.GcScriptGuiArray() guiText = GvVisionAssembly.GsScriptGuiText() guiText.sScriptGuiStyle = guiStyle guiText.strText ="text" guiText.posX = 0.0 guiText.posY = 0.0 guiText.deg = 0.0 guiArray.Add(guiText) GvGuiDataAgent.SetGraphicDisplay("View-1", guiArray)
