GsScriptGui3DProfileAreaResult

功能

要在视图窗口显示的轮廓面积。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
areaResult 要显示的轮廓面积。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

areaResult

  • 功能：要显示的轮廓图像，scGui3DProfileAreaResult类型.

注意

无

示例代码

```python gui = GvTool.GetToolData(“轮廓面积工具_006.面积GUI”)

Gui风格

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.nLineStyle = 0 # 线型 0:实线 1:虚线 2:点线 guiStyle.nLineWidth = 2 # 线宽 guiStyle.clrLineColor = [255, 0, 0]

Gui数组

guiArray = GvVisionAssembly.GcScriptGuiArray()

Gui轮廓面积

guiArea = GvVisionAssembly.GsScriptGui3DProfileAreaResult() guiArea .areaResult = gui guiArea .sScriptGuiStyle = guiStyle

将面积Gui添加到Gui数组

guiArray.Add(guiArea )

将Gui数组添加到View显示

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
