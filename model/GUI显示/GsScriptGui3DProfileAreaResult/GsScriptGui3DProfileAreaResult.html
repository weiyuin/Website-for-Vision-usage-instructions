GsScriptGui3DProfileAreaResult

Chức năng

Hiển thị diện tích biên dạng trong cửa sổ xem.

Giao diện

Phân loại Tên giao diện Mô tả giao diện
Thuộc tính sScriptGuiStyle Kiểu hiển thị, bao gồm font chữ, kích thước, màu sắc, v.v.
areaResult Diện tích biên dạng cần hiển thị.

Giải thích chi tiết

sScriptGuiStyle

  • Chức năng: Kiểu hiển thị, thuộc kiểu GsScriptGuiStyle.

areaResult

  • Chức năng: Biên dạng ảnh cần hiển thị, thuộc kiểu scGui3DProfileAreaResult.

Lưu ý

Không có

Mã ví dụ

```python gui = GvTool.GetToolData(“轮廓面积工具_006.面积GUI”)

Kiểu hiển thị GUI

guiStyle = GvVisionAssembly.GsScriptGuiStyle()
guiStyle.nLineStyle = 0 # Kiểu đường: 0 - liền, 1 - đứt, 2 - chấm
guiStyle.nLineWidth = 2 # Độ dày đường
guiStyle.clrLineColor = [255, 0, 0]

Mảng GUI

guiArray = GvVisionAssembly.GcScriptGuiArray()

GUI diện tích biên dạng

guiArea = GvVisionAssembly.GsScriptGui3DProfileAreaResult()
guiArea.areaResult = gui
guiArea.sScriptGuiStyle = guiStyle

Thêm GUI diện tích vào mảng GUI

guiArray.Add(guiArea )

Hiển thị mảng GUI trong View

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
