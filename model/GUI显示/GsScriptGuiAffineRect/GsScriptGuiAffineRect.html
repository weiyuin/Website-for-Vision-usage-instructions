GsScriptGuiAffineRect

Chức năng

Hiển thị hình chữ nhật affine trong cửa sổ xem.

Giao diện

Phân loại Tên giao diện Mô tả giao diện
Thuộc tính sScriptGuiStyle Kiểu hiển thị, bao gồm font chữ, kích thước, màu sắc, v.v.
affineRect Hình chữ nhật affine cần hiển thị.

Giải thích chi tiết

sScriptGuiStyle

  • Chức năng: Kiểu hiển thị, thuộc kiểu GsScriptGuiStyle.

affineRect

  • Chức năng: Hình chữ nhật affine cần hiển thị, thuộc kiểu scAffineRect.

Lưu ý

Không có.

Mã ví dụ

```python affinerect = GvTool.GetToolData(“几何定位_024.搜索位置”)

Kiểu hiển thị GUI

guistyle = GvVisionAssembly.GsScriptGuiStyle()
guistyle.nLineWidth = 10
guistyle.clrLineColor = [0, 255, 0]

Đối tượng GUI hình chữ nhật affine

affinerect_gui = GvVisionAssembly.GsScriptGuiAffineRect()
affinerect_gui.sScriptGuiStyle = guistyle
affinerect_gui.affineRect = affinerect

Mảng GUI

guiarray = GvVisionAssembly.GcScriptGuiArray()
guiarray.Add(affinerect_gui) # Thêm đối tượng GUI vào mảng GUI

Thêm mảng GUI vào cửa sổ xem

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray) ```
