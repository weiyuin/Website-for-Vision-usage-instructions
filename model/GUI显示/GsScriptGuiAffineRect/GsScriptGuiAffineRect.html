GsScriptGuiAffineRect

功能

要在视图窗口显示的仿射矩形。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
affineRect 要显示的仿射矩形。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

affineRect

  • 功能：要显示的仿射矩形，scAffineRect类型。

注意

无。

示例代码

```python affinerect = GvTool.GetToolData(“几何定位_024.搜索位置”)

Gui风格

guistyle = GvVisionAssembly.GsScriptGuiStyle() guistyle.nLineWidth = 10 guistyle.clrLineColor = [0, 255, 0]

仿射矩形Gui对象

affinerect_gui = GvVisionAssembly.GsScriptGuiAffineRect() affinerect_gui.sScriptGuiStyle = guistyle affinerect_gui.affineRect = affinerect

Gui数组

guiarray = GvVisionAssembly.GcScriptGuiArray() guiarray.Add(affinerect_gui) # 将Gui对象添加到Gui数组

将Gui数组添加到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray) ```
