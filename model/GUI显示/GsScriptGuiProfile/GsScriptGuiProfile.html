GsScriptGuiProfile

功能

要在视图窗口显示的轮廓图像。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
profile 要显示的轮廓图像。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

profile

  • 功能：要显示的轮廓图像，GsProfileData类型

注意

  1. 脚本设置轮廓图像时可以不用设置显示风格。

  2. 脚本工具中若添加了多个轮廓图像，只显示第一个轮廓图像。

  3. 如果决定使用脚本向视图中添加轮廓图像，则禁止手动向视图中添加轮廓图像。反之亦是。

示例代码

```python

Gui数组

guiArray = GvVisionAssembly.GcScriptGuiArray()

Gui轮廓

guiProfile = GvVisionAssembly.GsScriptGuiProfile() guiProfile.profile = GvTool.GetToolData(“图像文件夹_003.输出轮廓图像”)

将轮廓Gui添加到Gui数组

guiArray.Add(guiProfile)

将Gui数组添加的View显示

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
