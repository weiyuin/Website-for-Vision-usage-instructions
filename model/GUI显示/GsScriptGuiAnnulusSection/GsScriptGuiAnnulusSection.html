GsScriptGuiAnnulus

功能

要在视图窗口显示的圆环。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
annulusSection 要显示的圆环段。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

annulusSection

  • 功能：要显示的圆环段，scAnnulusSection类型

注意

无

示例代码

```python

Gui风格

guistyle = GvVisionAssembly.GsScriptGuiStyle() guistyle.nLineWidth = 2 guistyle.clrLineColor = [255, 255, 0]

圆环段Gui对象

center = GvVisionAssembly.sc2Vector(0, 0) startPhi = GvVisionAssembly.scRadian(0.0) spanPhi= GvVisionAssembly.scRadian(1.50) annulusSection_temp=GvVisionAssembly.scAnnulusSection() annulusSection_temp.SetCenterRadiusStartPhiSpanPhiRadialScale(center,50.0,startPhi,spanPhi, 2.0)

guiAnnulusSection = GvVisionAssembly.GsScriptGuiAnnulusSection() guiAnnulusSection.sScriptGuiStyle = guistyle guiAnnulusSection.annulusSection = annulusSection_temp

Gui数组

guiarray = GvVisionAssembly.GcScriptGuiArray() guiarray.Add(guiAnnulusSection) # 将Gui对象添加到Gui数组

将Gui数组设置到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray)

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray) ```
