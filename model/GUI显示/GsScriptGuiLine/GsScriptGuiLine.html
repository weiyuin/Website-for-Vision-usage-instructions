GsScriptGuiLine

Chức năng

Đường thẳng được hiển thị trong cửa sổ xem.

Giao diện

Phân loại Tên giao diện Mô tả giao diện
Thuộc tính sScriptGuiStyle Kiểu hiển thị, bao gồm phông chữ, kích thước, màu sắc, v.v.
line Đường thẳng cần hiển thị.

Giải thích chi tiết

sScriptGuiStyle

  • Chức năng: Kiểu hiển thị, kiểu dữ liệu GsScriptGuiStyle.

line

  • Chức năng: Đường thẳng cần hiển thị, kiểu dữ liệu scLine.

Lưu ý

Không có.

Ví dụ mã

```python

Kiểu hiển thị Gui

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.nLineStyle = 0 ## Kiểu đường: 0 - liền, 1 - đứt, 2 - chấm guiStyle.nLineWidth = 2 ## Độ dày đường guiStyle.clrLineColor = [255, 0, 0]

Mảng Gui

guiArray = GvVisionAssembly.GcScriptGuiArray()

Đường thẳng Gui

guiLine = GvVisionAssembly.GsScriptGuiLine() line = GvTool.GetToolData(“找线工具_002.直线结果”) guiLine.line = line guiLine.sScriptGuiStyle = guiStyle

Thêm đường thẳng Gui vào mảng Gui

guiArray.Add(guiLine)

Hiển thị mảng Gui trong View-1

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray)

```
