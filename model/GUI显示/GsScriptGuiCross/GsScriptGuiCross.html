GsScriptGuiCross

Chức năng

Dùng để hiển thị dấu thập trong cửa sổ xem.

Giao diện

Phân loại Tên giao diện Mô tả giao diện
Thuộc tính sScriptGuiStyle Kiểu hiển thị, bao gồm phông chữ, kích thước, màu sắc, v.v.
cross Vị trí hiển thị dấu thập.

Giải thích chi tiết

sScriptGuiStyle

  • Chức năng: Kiểu hiển thị, kiểu dữ liệu GsScriptGuiStyle.

cross

  • Chức năng: Vị trí hiển thị dấu thập, kiểu dữ liệu sc2Vector.

Lưu ý

Không có.

Ví dụ mã

```python

Thiết lập kiểu hiển thị GUI

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.bVisible = True guiStyle.nLineStyle = 2 guiStyle.nLineWidth = 2 guiStyle.clrLineColor = [0, 0, 255] guiStyle.bLabelVisible = True guiStyle.strLabelFont = "Arial" guiStyle.lFontSize = 100

Lấy mảng hiển thị GUI

guiArray = GvVisionAssembly.GcScriptGuiArray()

Cài đặt hiển thị dấu thập GUI

guiCross = GvVisionAssembly.GsScriptGuiCross() guiCross.sScriptGuiStyle = guiStyle guiCross.cross.SetX(500.0) guiCross.cross.SetY(200.0) guiArray.Add(guiCross)

Hiển thị mảng GUI trong View

GvGuiDataAgent.SetGraphicDisplay("View-1", guiArray) ```
