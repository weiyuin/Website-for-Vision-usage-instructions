GsScriptGuiStyle

功能

表示要在视图窗口显示的GUI风格，包括是否可见，字体大小，线型，线宽，颜色，标签是否可见，标签内容，标签字体等。

接口

分类 接口名称 接口描述
属性 bVisible GUI是否可见
nLineStyle 线型
nLineWidth 线宽
clrLineColor 颜色
bLabelVisible 标签是否可见
strLabel 标签内容
strLabelFont 标签字体
lFontSize 标签大小

详细说明

bVisible

  • 功能：GUI是否可见。布尔类型，True: 可见; False: 不可见。

nLineStyle

  • 功能：GUI线型，线型范围只支持0，1，2，3，4。

  • 0：表示实线 “————”。

  • 1：表示虚线 “————-”。

  • 2：表示点构成的线 “…………..”。

  • 3：表示下划线和点构成的线 “...._.”。

  • 4：表示下滑线和双点构成的线 “........_”。

nLineWidth

  • 功能：表示线宽，线宽设置范围是1\~15。

clrLineColor

  • 功能：表示颜色，List类型[r, g, b]。例如设置红色clrLineColor = [255, 0, 0]。

bLabelVisible

  • 功能：表示标签是否可见。布尔类型True或者False。

strLabel

  • 功能：设置标签内容，str类型。

strLabelFont

  • 功能：标签字体，str类型。例如 strLabelFont = “Arial”

lFontSize

  • 功能：字体大小(字号)，字体大小设置范围是8\~360整型。

注意

无。

示例代码

python guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.bVisible = True guiStyle.nLineStyle = 1 guiStyle.nLineWidth = guiStyle.clrLineColor = [255, 0, 0] guiStyle.bLabelVisible = True guiStyle.strLabel = "标签" guiStyle.strLabelFont = "Arial" guiStyle.lFontSize = 20
