GsScriptGuiPolyline

功能

要在视图窗口显示的多边形，Blob工具可能会用。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
polyline 要显示的多边形。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

polyline

  • 功能：要显示的多边形，scPolyline类型

注意

无。

示例代码

```python

设置GUI格式

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.bVisible = False guiStyle.nLineStyle = 3 guiStyle.nLineWidth = 2 guiStyle.clrLineColor = [255,0,0] guiStyle.bLabelVisible = True guiStyle.strLabelFont = “宋体” guiStyle.lFontSize = 10

### GUI设置

获取GUI显示数组

guiArray = GvVisionAssembly.GcScriptGuiArray()

多边形GUI

guiPolyline = GvVisionAssembly.GsScriptGuiPolyline() p0 = GvVisionAssembly.sc2Vector(149,233) p1 = GvVisionAssembly.sc2Vector(103,178) p2 = GvVisionAssembly.sc2Vector(100,99) p3 = GvVisionAssembly.sc2Vector(200,99) p4 = GvVisionAssembly.sc2Vector(200,200)

添加构成多边形的点

VectorVec = GvVisionAssembly.sc2VectorVec() VectorVec.append(p0) VectorVec.append(p1) VectorVec.append(p2) VectorVec.append(p3) VectorVec.append(p4) polyline0 = GvVisionAssembly.scPolyline(VectorVec,True)

guiPolyline.polyline = polyline0 guiStyle.strLabel = “多边形” #如果不需要显示label标签，这一行代码可以删除 guiPolyline.sScriptGuiStyle = guiStyle

向GUI显示数组添加GUI

guiArray.Add(guiPolyline)

将GUI数组设置到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
