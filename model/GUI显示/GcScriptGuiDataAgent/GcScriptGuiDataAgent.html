GcScriptGuiDataAgent

功能

用来将GcScriptGuiArray中存储的GUI对象数组显示到视图窗口，禁止自己定义一个GcScriptGuiDataAgent对象，内部已经定义了GvGuiDataAgent对象，可以直接使用。

接口

分类 接口名称 接口描述
构造 GcScriptGuiDataAgent GcScriptGuiDataAgent(strName, strID)构造函数。
函数 SetGraphicDisplay(str, value) 将要显示的GUI对象添加到视图窗口。
函数 AddGraphicToContainer(nIndex, value) 将要显示的GUI对象添加到自定义工具的图形数组输出中。

详细说明

GcScriptGuiDataAgent(strName, strID)

  • 功能：构造GcScriptGuiDataAgent对象。

  • 参数：

    • strName: str类型，当前工程名。例如”工程-1”。

    • strID: str类型，脚本工具ID。

  • 返回值：无

SetGraphicDisplay(str, value)

  • 功能：将要显示的GUI对象数组添加到视图窗口。

  • 参数：

    • str: str类型，视图名称。例如”View-1”。

    • value: GcScriptGuiArray类型，要添加的GUI对象数组。

  • 返回值：bool类型，表示是否添加成功。

    AddGraphicToContainer(nIndex, value)

  • 功能：将要显示的GUI对象添加到自定义工具的图形数组输出中。

  • 参数：

    • nIndex: int类型，图形数组的索引，范围为[0,5]。例如:0。

    • value: GcScriptGuiArray类型，要添加的GUI对象数组。

  • 返回值：bool类型，表示是否添加成功。

注意

  1. 禁止自己定义一个GcScriptGuiDataAgent对象，内部已经定义了GvGuiDataAgent对象，可以直接使用。
  2. 自定义工具内部脚本工具不能使用SetGraphicDisplay函数，只有普通工程里面可以使用。
  3. 普通工程中脚本工具不能使用AddGraphicToContainer，只有自定义工具内部可以使用。

示例代码

1、普通工程内部添加GUI

python guiArray = GvVisionAssembly.GcScriptGuiArray() guiText = GvVisionAssembly.GsScriptGuiText() guiText.strText ="text" guiText.posX = 0.0 guiText.posY = 0.0 guiText.deg = 0.0 guiArray.Add(guiText) GvGuiDataAgent.SetGraphicDisplay("View-1", guiArray) 2、自定义工具内部添加GUI

python guiArray = GvVisionAssembly.GcScriptGuiArray() guiText = GvVisionAssembly.GsScriptGuiText() guiText.strText ="text" guiText.posX = 0.0 guiText.posY = 0.0 guiText.deg = 0.0 guiArray.Add(guiText) GvGuiDataAgent.AddGraphicToContainer(0, guiArray) GvGuiDataAgent.AddGraphicToContainer(1, guiArray)
