GsScriptGuiPointsDis

功能

要在视图窗口显示的两点距离。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
pointsDis 要显示的两点距离。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

pointsDis

  • 功能：要显示的轮廓图像，scGuiPointsDistance类型。

注意

无

示例代码

```python gui = GvTool.GetToolData(“点云尺寸测量工具_004.尺寸线段GUI”)

Gui风格

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.nLineStyle = 0 ## 线型 0:实线 1:虚线 2:点线 guiStyle.nLineWidth = 2 ## 线宽 guiStyle.clrLineColor = [255, 0, 0]

Gui数组

guiArray = GvVisionAssembly.GcScriptGuiArray()

Gui两点距离

guiDis = GvVisionAssembly.GsScriptGuiPointsDis() guiDis.pointsDis = gui guiDis.sScriptGuiStyle = guiStyle

将轮廓Gui添加到Gui数组

guiArray.Add(guiDis)

将Gui数组添加到View显示

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
