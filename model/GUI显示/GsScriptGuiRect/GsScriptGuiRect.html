GsScriptGuiRect

功能

要在视图窗口显示的矩形。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
rect 要显示的矩形。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

rect

  • 功能：要显示的矩形，scRect类型

注意

无。

示例代码

```python

设置GUI格式

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.bVisible = False guiStyle.nLineStyle = 3 guiStyle.nLineWidth = 2 guiStyle.clrLineColor = [255,0,0] guiStyle.bLabelVisible = True guiStyle.strLabelFont = “宋体” guiStyle.lFontSize = 10

########## GUI设置 ################

获取GUI显示数组

guiArray = GvVisionAssembly.GcScriptGuiArray()

矩形GUI

guiRect = GvVisionAssembly.GsScriptGuiRect() rect0 =GvVisionAssembly.scRect(GvVisionAssembly.sc2Vector(100,100),GvVisionAssembly.sc2Vector(200,300))

guiRect.rect = rect0 guiStyle.strLabel = “矩形” guiRect.sScriptGuiStyle = guiStyle guiArray.Add(guiRect)

将GUI数组设置到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
