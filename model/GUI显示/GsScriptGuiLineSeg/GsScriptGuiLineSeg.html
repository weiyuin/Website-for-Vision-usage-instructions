GsScriptGuiLineSeg

功能

要在视图窗口显示的线段。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
lineSeg 要显示的线段。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

lineSeg

  • 功能：要显示的直线，scLineSeg类型

注意

无。

示例代码

```python

Gui风格

guiStyle = GvVisionAssembly.GsScriptGuiStyle() guiStyle.nLineStyle = 0 ## 线型 0:实线 1:虚线 2:点线 guiStyle.nLineWidth = 2 ## 线宽 guiStyle.clrLineColor = [255, 0, 0]

Gui数组

guiArray = GvVisionAssembly.GcScriptGuiArray()

Gui线段

guiLineSeg = GvVisionAssembly.GsScriptGuiLineSeg() lineSeg = GvTool.GetToolData(“找线工具_002.线段结果”) guiLineSeg.lineSeg = lineSeg guiLineSeg.sScriptGuiStyle = guiStyle

将直线Gui添加到Gui数组

guiArray.Add(guiLineSeg)

将Gui数组添加的View-1显示

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiArray) ```
