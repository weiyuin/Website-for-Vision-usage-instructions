GsScriptGuiEllipse

功能

要在视图窗口显示的椭圆。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
ellipse 要显示的椭圆。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

ellipse

  • 功能：要显示的椭圆，scEllipse类型

注意

无。

示例代码

```python ellipse = GvTool.GetToolData(“椭圆拟合工具_002.椭圆结果”)

Gui风格

guistyle = GvVisionAssembly.GsScriptGuiStyle() guistyle.nLineWidth = 5 guistyle.clrLineColor = [0, 255, 0]

椭圆Gui对象

ellipse_gui = GvVisionAssembly.GsScriptGuiEllipse() ellipse_gui.sScriptGuiStyle = guistyle ellipse_gui.ellipse = ellipse

Gui数组

guiarray = GvVisionAssembly.GcScriptGuiArray() guiarray.Add(ellipse_gui) ## 将Gui对象添加到Gui数组

将Gui数组设置到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray) ```
