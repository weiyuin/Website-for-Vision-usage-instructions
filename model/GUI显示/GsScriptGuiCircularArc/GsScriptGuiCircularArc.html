GsScriptGuiCircularArc

功能

要在视图窗口显示的圆弧。

接口

分类 接口名称 接口描述
属性 sScriptGuiStyle 显示风格，包括字体，大小，颜色等。
circularArc 要显示的圆弧。

详细说明

sScriptGuiStyle

  • 功能：显示风格，GsScriptGuiStyle类型。

circularArc

  • 功能：要显示的圆弧，scCircularArc类型

注意

无

示例代码

```python circularArc = GvTool.GetToolData(“找圆工具_023.圆弧结果”)

Gui风格

guistyle = GvVisionAssembly.GsScriptGuiStyle() guistyle.nLineWidth = 2 guistyle.clrLineColor = [0, 255, 0]

创建圆弧Gui

circularArc_gui = GvVisionAssembly.GsScriptGuiCircularArc() circularArc_gui.sScriptGuiStyle = guistyle circularArc_gui.circularArc = circularArc

将圆弧Gui添加到Gui数组中

guiarray = GvVisionAssembly.GcScriptGuiArray() guiarray.Add(circularArc_gui)

将Gui数组添加到视图

GvGuiDataAgent.SetGraphicDisplay(“View-1”, guiarray) ```
