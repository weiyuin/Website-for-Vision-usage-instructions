short转string适配器工具

功能

将short类型的数据，转换成两字节字符串。如short型数据12866，对应的十六进制为3242H，高低位调整选择“是”，则工具的转换结果为“2B”，高地位调整选择“否”，则工具的转换结果为“B2”。其中十六进制的32对应ASCII表中的字符2，十六进制的42对应ASCII表中的B。

参数

分类 参数名称 参数描述
属性窗口 高低位调整 是：高位在前；否：低位在前。
图像窗口 无
数据链 输入数据 需要转换的字符串。
高级界面 无

输出

分类 参数名称 参数描述
监视窗口 输出数据 输出转换之后的字符串结果
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 输出数据 输出转换之后的字符串结果，供其他工具使用。同监视窗口输出数据参数。
