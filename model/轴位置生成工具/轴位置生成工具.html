轴位置生成

功能

本工具可将三个double型的坐标数据转换为GcPlatformAxisPos类型的轴位置数据结果。该轴位置结果通常作为标定轴位置获取工具的初始轴位置数据链输入。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 轴X位置 double型的轴位置X坐标分量
轴Y位置 double型的轴位置Y坐标分量
轴D位置 double型的轴位置D坐标分量
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 平台轴位置 GcPlatformAxisPos型的平台轴位置结果
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无 无
数据链 平台轴位置 GcPlatformAxisPos型的平台轴位置结果，供后序工具使用，同监视窗口

详细说明

无

注意

  • 该类型转换工具的主要应用场景是将通信接收工具接收到的double型坐标数据转换为轴坐标数据，从而作为标定轴位置获取工具的初始轴位置输入。
