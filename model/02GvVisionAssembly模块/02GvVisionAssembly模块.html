GvVisionAssembly模块

该模块提供了访问和操作工具的方法，包含了工具涉及到的一些数据类型，例如sc2Vector,GcCoordPos,GcPlatformAxisPos等。

  • GvVisionAssembly.GcCoordPos

    表示位置姿态。

  • GvVisionAssembly.GcPlatformAxisPos

    表示轴位置。

  • GvVisionAssembly.GcScriptGuiArray

    要显示到视图的GUI数组。添加的GUI对象，如果想要显示到视图窗口，必须添加到GcScriptGuiArray数组。

  • GvVisionAssembly.GcScriptGuiDataAgent

    视图窗口代理。GcScriptGuiArray需要通过该代理将GUI对象添加到视图窗口。

  • GvVisionAssembly.GsScriptGuiText

    文本GUI。

  • GvVisionAssembly.GsScriptGuiCross

    十字GUI。

  • GvVisionAssembly.GsScriptGuiRect

    矩形GUI。

  • GvVisionAssembly.GsScriptGuiPolyline

    多边形GUI。

  • GvVisionAssembly.GsScriptGuiCircle

    圆形GUI。

  • GvVisionAssembly.GsScriptGuiStyle

    GUI风格。

  • GvVisionAssembly.GeMsgReportType

    消息报告类型。

  • GvVisionAssembly. ReportMessage

    生成消息报告，根据不同的消息报告类型，在对应位置生成消息，例如警告、错误消息。

    ```python

    消息报告函数，相当于消息报告工具。

    strMsg: 要输出的消息；

    eType: 消息类型，参见GeMsgReportType的说明。

    bIsForbid: 是否禁止弹窗，仅对有弹窗的消息类型有作用。True:禁止弹窗, False:允许弹窗。

    GvVisionAssembly.ReportMessage(str strMsg, GeMsgReportType eType, boolbIsForbid) ```

  • GvVisionAssembly.sc2Vector

    二维向量，表示定位结果的搜索坐标或者找圆结果的圆心。

  • GvVisionAssembly.sc2VectorVec

    二维向量的std::vector形式。如果要获取其中的元素，需要使用__getitem__(index)，index从0开始，例如获取第二个元素。

    python vecGeomResult = GvTool.GetToolData("几何定位_001.搜索坐标数组") geomResult0 = vecGeomResult.__getitem__(1)

  • GvVisionAssembly.sc2XformLinear

    二维线性变换，通常出现在定位工具的输出参数中。

  • GvVisionAssembly.scRadian

  • GvVisionAssembly.scRect

    表示矩形，例如几何定位的搜索区域，Blob工具的矩形ROI区域。

  • GvVisionAssembly.scAffineRect

    表示仿射矩形，例如几何定位的训练区域，Blob工具的仿射矩形ROI区域。

  • GvVisionAssembly.scCircle

    表示圆形，例如找圆工具的圆结果，Blob工具的圆形ROI区域。

  • GvVisionAssembly.scCircularArc

    表示圆弧。

  • GvVisionAssembly.scAnnulus

    表示圆环。

  • GvVisionAssembly.scEllipse

    表示椭圆，例如Blob工具的椭圆ROI区域，灰度检测工具的椭圆检测区域。

  • GvVisionAssembly.scLine

    表示直线，例如找线工具的找线结果。

  • GvVisionAssembly.scLineSeg

    表示线段。

  • GvVisionAssembly.scPolyline

    表示多边形，例如Bolb工具的多边形ROI。
