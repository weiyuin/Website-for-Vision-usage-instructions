基准修正标定计算工具

功能

基准修正标定计算工具是修改标定结果中的基准，即对标定结果中的图像坐标和平台坐标进行修改。修正的平台坐标是由工具输入的图像坐标向量根据原始标定结果计算得出，即将图像坐标转换为平台坐标。该工具的输入“图像坐标向量”是基准点的图像坐标，类型是GcCoordPos类型。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 图像坐标向量 新基准点的图像坐标，类型为GcCoordPos。
标定结果 待修改的原始标定结果。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 修正后标定结果 修改基准点后的标定结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 修正后标定结果 输出修改基准点后的标定结果，供其他工具使用，同监视窗口“修正后标定结果”参数。
