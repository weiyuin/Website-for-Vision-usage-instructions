多区域振镜位置计算工具

功能

多区域振镜位置计算工具是将输入的图像坐标经过计算后，得到转换后的振镜坐标，适用于将图像划分为多个小区域进行关联的场景。

参数

分类 参数名称 参数描述
属性窗口 产品坐标系补偿值 输入产品坐标系下的修正补偿值，包括X偏移向量、Y偏移向量和旋转角度。
是否忽略角度变化 控制转换计算过程中是否忽略转换图像坐标的角度变化，选择“是”，计算过程中角度值设为0。
是否采用多基准 选择“是”，转换计算过程中，利用距离转换图像坐标最近的辅助点来计算对应的振镜坐标；选择“否”，利用单个标定区域的中间点(第5个辅助点)来计算对应的振镜坐标。
是否采用绝对角度 采用绝对角度指将输出振镜坐标的旋转角度规范化到[-PI/2,PI/2]。
是否采用最近组标定 控制转换计算过程中是否采用最近组转换矩阵进行标定，选择“是”，根据最近组转换矩阵进行标定；选择“否”，根据输入的标定组号对应的转换矩阵进行标定。
图像窗口 无 无
数据链 输入图像坐标 输入需要转换的图像坐标。
X轴方向 振镜X坐标轴方向。
Y轴方向 振镜Y坐标轴方向。
二维线性变换 输入图像坐标与振镜坐标关联后的矩阵信息，从多区域标定工具链接得到。
图像坐标 输入图像坐标与振镜坐标关联后图像坐标点集，从多区域标定工具链接得到。
激光坐标 输入的激光坐标点集，从多区域标定工具链接得到。
标定组数 是否采用最近组标定参数选择“否”时，需要配置该参数，int类型，取值范围1~标定组数。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出激光坐标 输出转换后的激光坐标。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出激光坐标 输出转换后的激光坐标，供后序工具使用，同监视窗口输出激光坐标参数。
