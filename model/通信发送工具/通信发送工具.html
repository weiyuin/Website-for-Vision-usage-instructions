通信发送工具

功能

通信发送工具主要是通过对目标设备端口发送特定格式的数据指令，完成与外部设备的信息交互。通常在使用之前需要在通信管理对话框（工具条-设置-通信）中完成对通信端口的设置。

参数

分类 参数名称 参数描述
属性窗口 通信端口 选择所需要通信的设备（由通信管理对话框完成端口配置）。
指令格式 将数据链中的发送数据进行格式转换（同字符串格式化工具）。
通信超时 设备通信超时等待时间（最小5ms，设置-1表示一直等待）。
图像窗口 无 无
数据链 发送数据 需要发送的数据，通常先由数据包生成工具处理。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 执行结果 工具执行结果。
执行时间 工具执行时间。
发送字符串 工具实际发送出去的数据
图像窗口 无 无
数据链 无 无

详细说明

无

注意

  • 在格式转换中，对于bool型变量是转换成0或1发送出去的，对应转换符号%d。
