基准图像坐标修正工具

功能

基准图像坐标修正工具的功能是修改标定结果中的图像坐标，将标定结果中的图像坐标X改为“图像坐标X修正值”，图像坐标Y改为“图像坐标Y修正值”。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 原始标定结果 待修改的标定结果。
图像坐标X修正值 新的图像坐标X，输入double型，可以通过自己定义变量获得。
图像坐标Y修正值 新的图像坐标Y，输入double型，可以通过自己定义变量获得。
高级界面 无 无

输出

分类 参数名称 参数描述（参数的含义）
监视窗口 修正后的标定结果 修改图像坐标之后的标定结果。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 修正后的标定结果 修改图像坐标之后的标定结果，供其他工具使用，同监视窗口“修正后的标定结果”参数。
