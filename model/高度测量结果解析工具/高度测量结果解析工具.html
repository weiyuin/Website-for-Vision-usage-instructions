高度测量结果解析工具

功能

高度测量结果解析工具主要用于对高度测量工具2、通用高度测量工具的测量结果进行解析，得到测量高度的最大值、最小值、均值和中值。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 高度测量结果 需要解析的高度结果。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 最大值 高度测量结果的最大值。
最小值 高度测量结果的最小值。
均值 高度测量结果的均值。
中值 高度测量结果的中值。
是否有效 高度测量结果是否有效，无效的话最大值、最小值、均值、中值都是错误的。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 最大值 输出最大值信息，供后序工具使用，同监视窗口的最大值参数。
最小值 输出最小值信息，供后序工具使用，同监视窗口的最小值参数。
均值 输出均值信息，供后序工具使用，同监视窗口的均值参数。
中值 输出中值信息，供后序工具使用，同监视窗口的中值参数。
是否有效 输出高度测量结果是否有效，供后序工具使用，同监视窗口的中值参数。
