最大最小值工具

功能

最大最小值工具可以用来求解一组输入数据中的最大值和最小值。

参数

分类 参数名称 参数描述
属性窗口 数据个数 在N个数据中求解最大值和最小值，N即为数据个数，取值范围为[1,9]，默认为2。
数据类型 输入数据的数据类型，最大最小值工具提供的数据类型分为2种：double和int。
图像窗口 无 无
数据链 输入数据0 一组数据中的第一个输入值。
输入数据1 一组数据中的第二个输入值。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 最大值 一组数据中的最大值。
最小值 一组数据中的最小值。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 最大值 一组数据中的最大值，供后序工具使用，同监视窗口的最大值参数。
最小值 一组数据中的最小值，供后序工具使用，同监视窗口的最小值参数。

详细说明

无

注意

  • 数据链窗口输入数据的类型和属性窗口的数据类型必须一致，否则数据链参数找不到来源，最大最小值工具不能执行。

  • 属性窗口的数据个数参数决定了数据链窗口输入数据的个数。
