位置姿态生成

功能

本工具可将sc2Vector型的位置坐标数据与double型旋转量组合转换为GcCoordPos类型的位置姿态数据结果。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 位置坐标 sc2Vector型位置坐标数据
旋转量 double型的旋转量
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 位置姿态 GcCoordPos型的位置姿态数据结果
执行结果 工具执行结果
执行时间 工具执行时间
图像窗口 无 无
数据链 位置姿态 GcCoordPos型的位置姿态数据结果，后序工具使用，同监视窗口

详细说明

无

注意

  • 该类型转换工具的主要应用在图像到平台坐标转换工具之前，将定位到的图像特征点坐标，并加上相应的旋转量，组合成位置姿态结果。该位置姿态结果作为图像到平台坐标转换工具的图像坐标输入。
