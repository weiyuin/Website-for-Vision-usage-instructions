GcCoordPos

功能

表示位置姿态，例如位置姿态生成工具的输出，标定结果（标准标定计算工具）的图像坐标和平台坐标。

接口

分类 接口名称 接口描述
构造 GcCoordPos GcCoordPos()默认构造函数，XYD初始化为0.0
GcCoordPos GcCoordPos (double,double,double)构造函数
GcCoordPos GcCoordPos (sc2Vector,double)构造函数
属性 X X坐标
Y Y坐标
D D坐标，角度

详细说明

GcCoordPos( )

  • 功能：构造GcCoordPos对象。

  • 参数：无

  • 返回值：无

GcCoordPos( double dPosX, double dPosY, double dAngle )

  • 功能：构造GcCoordPos对象。

  • 参数：

    dPosX：X坐标

    dPosY：Y坐标

    dAngle：D坐标，表示角度

  • 返回值：无

GcCoordPos( sc2Vector Pos, double dAngle )

  • 功能：构造GcCoordPos对象。

  • 参数：

    Pos：X，Y坐标

    dAngle：D坐标，表示角度

  • 返回值：无

X/Y/D

  • 功能：X坐标/Y坐标/D坐标。浮点型，可读写。
