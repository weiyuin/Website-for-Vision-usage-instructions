GcPlatformAxisPos

功能

表示机械轴位置，例如轴位置生成工具的输出，图像到平台坐标转换工具可能使用到的平台轴位置、相机轴位置，标定结果中的交互步长都是该类型。

接口

分类 接口名称 接口描述
构造 GcPlatformAxisPos GcPlatformAxisPos()默认构造函数，XYD初始化为0.0
GcPlatformAxisPos GcPlatformAxisPos (double,double,double)构造函数
属性 X X轴坐标
Y Y轴坐标
D 旋转轴坐标

详细说明

GcPlatformAxisPos( )

  • 功能：构造GcPlatformPos对象。

  • 参数：无

  • 返回值：无

GcPlatformAxisPos( double dPosX, double dPosY, double dAngle )

  • 功能：构造GcPlatformPos对象。

  • 参数：

    • dPosX：X轴坐标

    • dPosY：Y轴坐标

    • dAngle：旋转轴坐标，表示角度

  • 返回值：无

X/Y/D

  • 功能：X轴坐标/Y轴坐标/旋转轴坐标。浮点型，可读写。
