位置姿态转轴位置

功能

本工具可将GcCoordPos型位置姿态数据转换为GcPlatformAxisPos类型的轴位置数据。工具输入可以是图像到平台坐标转换工具的平台坐标输出等，工具的输出可以作为轴位置解析工具的输入。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 位置姿态 GcCoordPos型的位置姿态数据结果。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 轴位置 GcPlatformAxisPos型的轴位置数据。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 轴位置 GcPlatformAxisPos型的轴位置数据，供后序工具使用，同监视窗口。

详细说明

轴位置

GcPlatformAxisPos类型数据，该类型为三个double型坐标数据组成的结构体。在标定对位工程中，标定轴位置获取工具的初始轴位置输入与轴位置结果、交互运动步长、对位计算工具的对位偏移量结果等变量均为GcPlatformAxisPos类型。

位置姿态

GcCoordPos类型数据，该类型同样为三个double型坐标数据组成的结构体。在标定对位工程中，标定计算工具的图像坐标、平台坐标输出，图像到平台坐标转换工具的图像坐标输入与位置姿态输出均为GcCoordPos类型数据。

注意

  • 该工具将GcCoordPos型数据转换到GcPlatformAxisPos类型数据，可根据项目需要运用。
