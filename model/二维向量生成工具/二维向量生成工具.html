二维向量生成工具

功能

二维向量生成工具用来将两个double类型数据生成一个二维向量（scVector）。其作用与二维向量解析工具相反。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 X分量 二维向量的X分量。
Y分量 二维向量的Y分量。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 二维向量 由两个double类型数据转换而成的二维向量数据。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 二维向量 输出由两个double类型数据转换而成的二维向量数据，供后序工具使用，同监视窗口的二维向量参数。
