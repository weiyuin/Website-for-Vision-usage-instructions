二维坐标空间转换工具

功能

将坐标空间1中的点集通过“坐标空间转换关系”转换到另一个坐标空间2中。相当于。其中是输入的点，是转换后坐标空间中的点坐标，是从坐标空间1到坐标空间的坐标转换关系。

参数

分类 参数名称 参数描述
属性窗口 无 无
图像窗口 无 无
数据链 坐标空间1点集 待转换的坐标空间1中的点集。
坐标空间转换关系 坐标空间1到坐标空间2的转换关系。
高级界面 无 无

输出

分类 参数名称 参数描述（参数的含义）
监视窗口 坐标空间2点集 转换到坐标空间2中的点集。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 执行结果 显示工具执行结果，执行成功显示“OK”，执行失败显示“NG”，同监视窗口的执行结果参数。
数据链 坐标空间2点集 转换到坐标空间2中的点集，供其他工具使用。同监视窗口参数。

详细说明

无

注意

  • “二维坐标空间转换工具”的输入点集，可以是数据类型选择“sc2Vector”的数组生成工具的输出数组。

  • “二维坐标空间转换工具”的输出结果“坐标空间2点集”可以使用数组解析工具，数据类型选择sc2Vector，配合索引值就可以获取。参见“二维坐标空间转换.gvp”例程。
