随机数生成工具

功能

生成一组随机数。

参数

分类 参数名称 参数描述
属性窗口 数据类型 生成数组的数据类型，分为int型以及double型。
随机数组长度 生成的数组的长度，取值范围[0,2147483647]。
上限 取值为double，上限>下限。
下限 取值为double，下限<上限。
图像窗口 无 无
数据链 无 无
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 随机数数组 产生的由随机数组成的数组。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 随机数数组 产生的由随机数组成的数组。与监视窗口中的“随机数数组”是同一个参数。
