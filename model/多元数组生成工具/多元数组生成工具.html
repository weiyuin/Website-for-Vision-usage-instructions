多元数组生成工具

功能

多元数组生成工具可以用来将相同类型的多组独立的数据，生成多元数组。

参数

分类 参数名称 参数描述
属性窗口 数据类型 选择生成数组的数据类型，根据实际使用情况选择对应的数据类型。
元素个数 个数为2至64。
图像窗口 无 无
数据链 输入数据 输入数据（与属性窗口数据类型相同）。
高级界面 无 无

输出

分类 参数名称 参数描述
监视窗口 输出数组 输出生成的数组信息。
执行结果 工具执行结果。
执行时间 工具执行时间。
图像窗口 无 无
数据链 输出数组 输出生成的数组信息，供后序工具使用，同监视窗口的输出数组参数。

详细说明

多元数组生成工具提供的数据类型如下所示。

bool 布尔 int 整型
double 双精度浮点型 float 单精度浮点型
long 长整型 sc2Vector 二维向量
GcCalibrateResult 标定结果 GcCoordPos 位置姿态
GcPlatformAxisPos 轴位置 GcBoardPoints 靶标角点坐标
sc2XformLinear 二维线性变换 scCaliperOneEdgeResult 单边卡尺结果
scCaliperTwoEdgeResult 双边卡尺结果 scCircle 找圆工具结果
scGeomSearchResult 几何定位结果 scImage24 24位彩色图像
scImage8 8位灰度图像 scLine 找线工具结果
scPatSearchMaxResult PatSearchMax结果 scGcCoordPosVector 位置姿态向量
scSc2vectorVector 二维向量向量 scFeatureletVector 特征链
GsTextDisplayParam 文本显示参数 DefectResultInfo 瑕疵检测结果
DMSingleReaderResult 单个DM读码结果 GsGlueDetectResult 胶体位置检测结果
GsSingleBlobResult 单个Blob结果 GsGlueMeasureResult 胶体尺寸测量结果
scCrossSearchResult 十字定位结果 QRSingleReaderResult 单个QR读码结果
GsMultiCLOutputResult 多圆多线定位结果 SingleLinearCodeResult 一维读码结果
EdgeDeviation 边缘检测结果
